第五堂課

2026.03.18
本堂內容

一、本堂核心

這堂把三個練習連在一起：站立時感受重心位置、折胯蹲練的細節修正、坐姿的初步介紹。核心觀念是「鬆動是唯一的流動方式」——不是用力站起來，不是用力坐下去，而是找到哪一個啟動點放鬆，讓身體自然流動過去。打拳就是把這個狀態帶著走。

二、站立的重心覺察

站立時把注意力放在腳掌，讓重心前後輕輕移動，感受腳趾浮起和腳跟浮起的差異，再停在腳掌均勻接觸地面的中間位置。膝蓋微彎（不打直）讓重力往下走，而不是撐在上半身；打直的膝蓋會撐，久了就僵。

當你有放鬆的東西進來，我們膝蓋微彎的時候搭配身體的架構，它是會往下走。

站立時只要把注意力放在吐氣，讓上半身的重量沉下來，腳底就會變得比較穩。這不需要改變外型，只靠呼吸的觀察就能做到。

三、折胯蹲練：細節修正

最常見的問題是手和臀各自動，沒有形成整體平衡。正確概念是：臀部（髖關節）被往後拉，上半身為了平衡「自然地往前延伸」，不是手主動舉起來。後面出去多少，前面就補多少——這是流動，不是兩個分開的動作。

慢是關鍵。「切成十五等份」：從站立到最低點要切成十五個微小步驟，幾乎感覺不到在動，這樣才能看到每個過渡點的變化。起身時，膝蓋往前放鬆是「泄壓的出口」，讓積在腿部的力量流動出去，上半身就自然直立。

四、坐姿初步介紹

椅子放在背後一個腳掌距離。同樣用折胯的方式慢慢往後，屁股輕觸椅面後——停住，不要坐下去。這時候把手放下來，全部的壓力都在腿上，這是「充電」的狀態。接著從腳趾往上一層一層放鬆：腳趾 → 小腿 → 膝蓋 → 大腿。每放鬆一層，上半身就會微微地直立起來一點。最後把腳掌往前調到膝蓋正下方，讓腳掌、膝蓋、髖關節在一直線上，壓力完全往下走。坐骨+腳掌四個支撐點，上半身就完全沒有壓力了。

五、回家練習

  • 站立重心練習：腳掌前後輕移找中間，膝蓋微彎，注意力在吐氣讓上半身的重量往下走。
  • 折胯蹲練：慢慢做，切十五等份，感受屁股往後＋上半身往前的流動平衡；起身靠膝蓋往前泄壓。
  • 坐姿練習（剛接觸）：椅子一腳掌距離，折胯到輕觸，從腳趾往上一層一層放鬆，讓上半身透過腿的鬆而直立。

一、這堂課真正要練的是什麼

這堂有三個層次的練習：站立重心覺察、折胯蹲練的細節深化、以及坐姿的初步介紹。三者的底層邏輯是一致的：用鬆（而非用力）讓身體移動和穩定。

阿郎這堂說了一個關鍵的觀念：打太極拳，最終是要學到「用鬆的方式讓身體動」。不是手主動舉，而是某個地方放鬆之後它自然飄起來；不是腿用力撐，而是某個地方泄壓之後身體自然直立。「打拳就是把身體端著走的放鬆狀態」——這是這堂課的核心精神。

二、站立重心覺察：從腳底開始建立穩定

這是坐姿練習的前置基礎。

先讓腳掌與地面均勻接觸：重心略往前（腳跟浮起）、再略往後（腳趾浮起），然後停在腳掌均勻接觸地面的位置。這個中間位置不一定完全靜止，會有些微晃動，這是正常的——先知道在哪個範圍，不要求一開始就完全靜止。

膝蓋打直 vs 微彎的差異：打直的膝蓋會「撐」，讓力量往上積，久了會僵，而且撐不了太久。微彎讓重力往下走，搭配呼吸的放鬆，整個站姿就會穩。

當你有放鬆的東西進來，我們膝蓋微彎的時候搭配身體的架構，它是會往下走。

搭配吐氣：站立時不需要改變外型，只要把注意力放在吐氣，讓上半身的重量自然沉下來，腳底就會感覺到更踏實的重量。這就是前幾堂說過的「呼吸把重心往下帶」在站姿上的應用。

三、折胯蹲練：概念深化與常見問題修正

這堂補充了兩個重要說明：

整體流動，不是兩個分開的動作

最常見的問題是手和臀分開動——臀往後、手往上，變成兩個各自的動作。正確的概念是：臀部（髖關節）「被往後拉」，上半身為了平衡整個身體的重心，「自然地往前延伸」。延伸不是手主動舉起，而是重心被帶往後的時候，前面需要對等地輕輕送出。後面出去多少，前面就補多少——這是一個連動的流動，不是分開操作的。

速度切成十五等份

要慢到幾乎感覺不到自己在動。從站立到最低點的範圍，切成十五個微小步驟，每一步驟都要「踩到」、感受到，再繼續下一步。這樣才能看到每個過渡點的力量變化，才能調整。快了就什麼都看不到，直接坐下去，就失去練習的意義。

起身的「泄壓」邏輯

起身不是靠腿的力量直接站起來。這裡阿郎用了一個比喻：把壓力積在腿部之後，膝蓋往前是「戳一個洞讓力量流出去」的動作——讓積在那裡的力量有一個出口，身體就自然跟著流動起來。用腿直接站起來是慣性的肌肉力量；用放鬆的方式起身，是找到哪個地方泄壓，讓那個流動帶動整個起身的過程。

四、坐姿初步介紹：從折胯到坐骨支撐

坐姿是打拳的身體基礎，這堂是初步介紹，細節下次繼續。

準備 椅子放在背後約一個腳掌的距離。用折胯的方式慢慢往後走——不是蹲，是髖關節往後，上半身往前平衡。

觸椅但不坐下 當屁股輕觸椅面，要停住，不要把重量完全放在椅子上（那樣就是一般的坐下，尾椎被壓住，全部壓力散掉了）。這個輕觸的狀態是「充電」——讓腿部保持著壓力，接下來的放鬆才有作用。

一層一層往上放鬆 把手放下來，全部壓力都在腿上。接著從腳趾開始往上，一層一層觀想放鬆：

  1. 腳趾 → 腳掌放鬆
  2. 小腿放鬆
  3. 膝蓋放鬆
  4. 大腿放鬆

每放鬆一層，上半身就會微微地直立起來一點——不是靠上半身的力量挺直，而是腿部壓力釋放之後，重量往下走，上半身自然就浮起來了。這和小腿拉筋時「從小腿往上一層一層放鬆，拉筋越來越深入」是同一個邏輯。

最終支撐狀態 最後把腳往前放，讓腳掌、膝蓋、髖關節對齊在一直線上（像水管通順，不折到）。這時候：坐骨（兩個點）+ 腳掌（兩個面）= 四個支撐點，上半身完全沒有壓力，就像放在盤子上輕輕端著。

打拳就是把身體端著走的放鬆狀態。

五、找啟動點：太極拳移動的邏輯

阿郎說了一個貫穿這堂課所有練習的核心觀念：太極拳裡的每個動作，都不是靠力量「做出來」，而是靠「找到啟動點然後放鬆它」讓身體自然流動。

手起來不是舉手，是某個地方放鬆讓它飄起來。手收回來不是拉，是某個地方放鬆讓它靠過來。站起來不是靠腿撐，是膝蓋泄壓讓流動帶起身體。

這就是為什麼現在的練習都在強調「慢」——快了你只能用肌肉的慣性力量，永遠找不到那個啟動點。慢才能感覺到「哦，這個地方一鬆，身體就開始動了」。

六、學員問答重點

有學員說折胯做完腳踝很痠。阿郎說：力量聚集在腳踝，代表練習時高度太低，或力量太集中在那邊。解法：不要蹲那麼低，讓力量留在髖關節，而不是跑到腳踝。

有學員問起身時要先膝蓋往前還是尾椎先動。阿郎說：如果要分步驟，就是膝蓋往前 → 屁股往上收 → 脊椎往上延伸；如果你已經對身體有感，只要抓尾椎的移動其他就跟著，是更簡潔的路徑。

七、回家練習與下次銜接

  • 站立重心練習：腳掌前後輕移找中間，膝蓋微彎，注意力在吐氣讓上半身重量往下走。
  • 折胯蹲練：切十五等份，感受整體流動（不是分開動），起身靠膝蓋泄壓而非腿部撐起。
  • 坐姿（初步接觸）：椅子一腳掌距離，折胯輕觸，從腳趾往上一層一層放鬆，讓上半身透過腿的鬆而直立。

下次課程（第六堂）將繼續深化坐姿練習的細節，並開始帶入第一個拳架動作。
