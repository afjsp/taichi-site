第五堂課

2026.03.18
本堂內容

第五堂課的核心要點，延續對身體覺察的練習，重點在於如何透過重心調整、呼吸配合與「折胯」動作，達到全身的放鬆與能量流動。

一、 站立時的重心與穩定 (Standing & Stability)

  • 尋找腳掌平衡點：
  • 自我測試：將重心微微前後移動，感受壓力在腳尖與腳跟之間的變化。當重心過前腳跟會浮起，過後則前腳掌浮起。
  • 中心定位：在極端的前後移動中，找出壓力分佈最均勻、身軀最安定且不需要額外出力支撐的「中心點」。
  • 避免「僵化」的支撐：
  • 膝蓋微彎：膝蓋若完全打直（鎖死），會形成往上撐的僵硬感，壓力會卡在髖關節與膝蓋。
  • 重力下沉：讓膝蓋微彎但不鎖死，使上半身的重量能順利穿透膝關節，直接掉到腳底，產生「紮實」的穩定感。
  • 動態調整：即使維持同樣姿勢，內部也要有微小的「擴張與收縮」變動，像呼吸一樣持續流動，才不會陷入僵化（緊）。

二、 呼吸與意念的減壓運用 (Breathing & Intent)

  • 吐氣卸壓：
  • 練習時不理會吸氣，專注於吐氣（特別是後半段）。
  • 吐氣時配合意念將上半身的重量與壓力往下放，直到腳底。
  • 意念帶向尾椎：
  • 在吐氣尾聲，將念頭導向尾椎（尾巴），這能讓念頭靜下來，並讓上半身感覺變輕、下半身變穩。

三、 核心動作：折胯蹲與重心平衡 (The Folding Squat)

這是本堂課最細微的練習，旨在練習身體的協調與對稱平衡。

  • 準備：雙腳與肩膀同寬，腳趾朝前。
  • 動作步驟：
  • 折胯（核心機制）：想像髖關節處被一股力量往後拉，而不是單純膝蓋蹲低。
  • 膝蓋不動：在下蹲的過程中，膝蓋固定在原位（不往前頂也不往後縮），這是測驗身體壓力的關鍵。
  • 手部平衡：雙手被動地隨重心往前方延伸。手起的幅度取決於屁股往後移了多少，目的是為了維持重心留在原處（腳掌中心）而不偏移。
  • 極慢速練習：將動作拆解成像「幻燈片」或「蝸牛移動」般緩慢（如：一個角度停15秒），才能覺察到微小的肌肉變化與重心位移。
  • 起立步驟：
  • 不要用大腿蠻力往上頂。
  • 膝蓋微向前流動，配合尾椎放鬆下沉，脊椎自然向上拉拔，利用身體的協調性輕鬆站起。

四、 坐姿練習與卸壓 (Seated Posture)

這是一種「練功」的坐法，旨在體會上半身如放在盤子上般輕盈的狀態。

  • 進入坐姿：利用上述的折胯動作慢慢降低，直到屁股接觸到椅子但不坐下去（不卸掉壓力）。
  • 觀想放鬆：在腿部充滿張力的狀態下，依序觀想放鬆腳趾、小腿、膝蓋、肩頸、大腿。
  • 調整支撐線：輕輕抬腳調整位置，讓腳掌、膝蓋、髖關節（髖部）連成一線，使重力像水管疏通般直接卸到腳底與坐骨這四個支撐點上。
  • 練習目標：讓肩膀與頸部的壓力完全釋放，這種放鬆感應帶入日常生活中，而非僅限於練習時。

五、 練習提醒

  • 卡點與流動：若練習動作時感到「動不了」，代表該處有「卡點（僵化）」。練習就是不斷尋找「鬆的開關」，讓能量重新流動。
  • 打哈欠是好現象：這代表身體正在透過深度放鬆解除卡點。
  • 量力而為：不需要蹲太低，以「不產生局部僵硬」或「不引起疼痛」為前提進行微調。

課程要點（詳細版）

一、課程主軸

  • 建立腳掌重心感知的完整認識
  • 深化折胯蹲姿，加入腳掌均勻接觸的檢驗
  • 探討「用放鬆觸發移動」的概念——太極拳移動邏輯的核心

二、棒棒糖比喻——重心為什麼要低

【棒棒糖】糖果上圓下細，重心在上方 → 稍微偏一點就整個倒

→ 身體重心太高（聳肩、意念在頭部）= 棒棒塔：不穩

→ 重心往下走（意念在腹部、腳掌）= 重心低的陀螺：穩

【腳掌感知的意義】

站穩的根本在腳底，不在上半身的努力

傳統太極要先站樁一年，練的就是腳掌均勻接地的感知

「你上半身越放鬆，腳底就越重——這是對的方向。腳底重，代表重量真的下來了。」

三、膝蓋微彎 vs 打直的差異

【膝蓋打直】

髖關節和膝蓋都在撐 → 兩個壓力點 → 支撐力量是「硬撐」

初期不會晃，但時間久了容易僵化

【膝蓋微彎】

重力能往下走，讓腳掌接收重量

配合呼吸可以做鬆緊的微調（肌肉不是固定的，是在持續微變化）

「你打直的時候比較不會晃，是因為有東西在撐著你。但那個撐，是硬撐，撐久了一定會僵。微彎的時候，重量可以往下走，你才是真的站穩。」

【如何讓微彎不僵化】

膝蓋不是固定的，是可以做輕微的擴張和收縮的

讓膝蓋持續有微小的鬆緊變化 → 站久了也不容易累

這就是把第一堂的「鬆緊交替」帶進站姿的應用

四、腳掌均勻接觸地面——練習方法

【感受前後重心】

重心往前一點，感受腳跟有點輕浮

重心往後一點，感受前腳掌有點輕浮

找到中間：腳跟跟前腳掌都穩穩踩著，腳趾頭不需要用力抓地

「腳趾頭要準備抓地的那個感覺，要放掉。就是輕輕放著，像是跟地板融在一起。」

【搭配吸吐氣練習】

吐氣 → 上半身壓力往下 → 腳底感覺更踏實

吸氣 → 意念帶起腳部壓力 → 腳掌感覺輕一些

→ 練習讓腳掌的感覺「均勻但不沉重」

五、折胯蹲姿深化（雙人輔助練習）

【雙人輔助做法】

輔助者：站在被練習者前方，手放在對方膝蓋上固定（膝蓋不能動）

被練習者：

① 感覺有人從後面把你的髖關節往後拉

② 上半身整體往前，手輕輕往前做平衡

③ 確認腳掌依然均勻接觸地面

【常見問題：起身時膝蓋往前跑】

這是習慣性的「用大腿撐起來」的動作

正確起身：先找到膝蓋的位置，再讓重心往前一點，讓身體自然起來

「起來是結果，不是目的。你只要做到讓那個卡點放掉，身體自然就起來了。你不用去想要起來。」

【水管折到比喻——肩膀放鬆的角度】

肩膀有一個讓氣血暢通的角度，找到這個角度就是放鬆肩膀

太往前 → 這邊水管折到

太往後 → 那邊水管也折到

→ 在大約45度的位置，讓兩邊水管都不折，氣血才能流暢

六、「用放鬆觸發移動」——太極拳移動邏輯

【核心原理】

太極拳的每一個移動，都是從「放鬆某個地方」開始，而不是從「施力」開始

好像水壩泄洪：戳一個洞，水自然流出去

好像針灸放血：針扎進去，氣血自然流動

【實際應用：用放鬆讓起身發生】

不是讓腿用力把自己撐起來

而是：找到膝蓋這個「節點」，讓它輕輕往前一點，整個卡住的地方就鬆了

→ 身體自然地流動起來，起身只是結果

【卡點的概念】

練習過程中，放鬆的時候反而不會動了 = 發現了一個「卡點」

不要害怕卡點，這才是真正在練習的地方

「你放鬆的時候動不了，這是好事。這代表你找到了一個你平常用力的地方。你現在要做的，就是讓那個地方，可以不用力。」

七、練習時間管理——避免練緊

【原則】頻率多，單次時間短

帶著疲勞持續練習 = 練緊，不是練鬆

練到覺得膝蓋或腿部緊繃了，就要停下來放鬆

動一動、按摩一下，讓肌肉「歸零」再繼續

「我們要的是鬆。動作只是來協助我們調到鬆的狀態，它本身不是目的。你練到緊了，你練的不是太極拳。」

八、問答重點節錄

  • Q：我站的時候，感覺腳變成兩根鐵柱，又沉又重，是對的嗎？

A：非常好！上半身放鬆的時候，腳一定會重。這代表重量真的下來了。你再配合吸氣，讓那個重心稍微帶起一點點，腳就會輕一些。不需要消除那個重，只要讓它不是僵化在那裡。

  • Q：折胯時我感覺重心一直往後跑，很難保持均衡。

A：代表你屁股往後太多，前面補的不夠。你可以把這個動作切成四等份，每次只做四分之一，讓你有時間感受說前後是不是均衡了。

九、課後練習建議

  • 站立時：每天做幾次「前後搖擺感受重心」練習，找到均勻的中間點
  • 折胯：每天 3-5 次，每次都確認腳掌均勻接觸地面
  • 膝蓋微彎站立：試著在日常站立時保持膝蓋微彎，配合呼吸讓它不僵化
  • 卡點探索：遇到卡住動不了的地方，先記下來，下次課可以和老師討論
