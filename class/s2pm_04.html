第四堂課

2026.05.28
本堂內容

2026-05-28｜主題：站樁三個階段——骨架、肌肉、心態

一、本堂核心

本堂正式介紹站樁（無極式）的完整三個階段：第一層調骨架，第二層調肌肉，第三層調心態。這三層做完，才是真正意義上的「站好了」。站樁不是站著不動，而是在那個站的狀態裡，從粗到細一層一層讓身體自己去找平衡。先複習了拉筋（手臂往後翻、肩胛骨拉筋），再引入站樁的完整結構。

二、拉筋複習：肩胛骨系列

前面（手腕往後翻）：手微彎、手指左右延伸，手掌緩慢往後翻，在肩胛骨有感覺的時候停著呼吸。收下巴、後腦勺稍微往後，延展更明顯。

後面（肩胛骨往下）：一手放成某個角度，手掌背面往下，這樣肩胛骨就會被「撥開」——脊椎和肩胛骨之間的肌肉會延展開來。左右輪流做，讓肩胛骨周圍的肌肉有鬆緊鬆緊的變化，就會慢慢有彈性。

三、站樁第一層：骨架

雙腳平行站立，與肩同寬 → 膝蓋微彎（不打直，也不往內夾）→ 收小腹、收尾鋁（讓腰椎盡量直）→ 雙肩自然下垂，大拇指輕貼褲縫 → 收下巴 → 舌頂上顎 → 頂頭懸。

這一層讓身體架構盡量平衡，不歪斜，讓重力分布均勻。

四、站樁第二層：調肌肉

重心慢慢往前再往後，找到一個讓腳踝和小腿壓力最小的位置，停在那裡。然後從小腿開始，依序往上放鬆：小腿 → 膝蓋 → 大腿 → 臀部、腰部、背部 → 小腹、胸腔 → 肩膀 → 兩手臂 → 頸部 → 臉部肌肉。

每一個位置能調就調，調不動的先留著，知道它的狀況就已經有機會。

五、站樁第三層：調心態

眼睛往前看，不抓焦點，讓光線自然進來，像看大螢幕一樣。耳朵等聲音進來，不去抓。呼吸知道自己在呼吸，不刻意深呼吸，但知道深淺快慢。這是「等待而覺知」的狀態——什麼都不做，但保持清楚的知道。

意識不多事，只有保留「知道」的成分就好。看到緊，能調就調；僵化的，先看著它，知道自己在呼吸，它也會在變化。

六、回家練習

  • 肩胛骨拉筋（前、後各一組），在緊的邊界停著呼吸。
  • 站樁三層練習：先把骨架擺好，再調肌肉，最後進入等待覺知的心態。每次五到十分鐘，腳酸了就動一動再繼續。
  • 快和慢都可以用：慢是觀察工具，快是整合測試，先由慢入手。

2026-05-28｜主題：站樁三個階段——骨架、肌肉、心態

一、這堂課真正要練的是什麼

本堂正式進入站樁（無極式）的系統性介紹，分三個階段：骨架 → 肌肉 → 心態。這三層加在一起，才是完整的站樁狀態。它不是「站著不動」，而是一個從粗到細、從外到內的自我掃描與調整過程。課程前半也複習了肩胛骨拉筋，讓學員先打開肩背的緊繃，作為站樁前的準備。

阿郎也在本堂說明了「用意不用力」的概念：太極拳追求的是用念頭動身體，而不是肌肉硬撐——但這需要先把多餘的力量清掉，才有機會到達那個狀態。

二、拉筋複習：肩胛骨前後兩個方向

前面（手腕往後翻，扣肩胛骨）：

  • 手不打直，微彎放著，手指往左右延伸
  • 手掌緩慢往後翻，一環扣一環：手掌 → 手肘 → 肩關節 → 肩胛骨
  • 翻到肩胛骨有感覺時停下來，加上收下巴、後腦勺稍微往後，讓頸椎有延展空間
  • 在那個緊的邊界停著呼吸，吐氣時往下鬆一點點
  • 放回來時也要慢：先從手腕轉回，再肩關節，再慢慢放下

後面（手掌背面往下，撥開肩胛骨）：

  • 一手放成某個角度，手掌背面朝下往下移
  • 這樣肩胛骨和脊椎之間的肌肉會被延展開來，像被「撥開」的感覺
  • 左右輪流做，讓肩胛骨周圍有鬆緊鬆緊的節律，肌肉就會慢慢有彈性

快的做法（抖動）和慢的做法都可以用：快是讓肌肉快速變化，但你看不到細節；慢是讓你看到細節，是觀察工具。阿郎建議先慢，建立對肌肉的感知之後，再加快做整合。

三、站樁第一層：骨架調整

這是讓身體「站直」的基本架構，目的是讓重力分布盡量均勻，減少不必要的壓力。

調整步驟：

  • 雙腳平行站立，與肩同寬（腳跟稍微往外提起轉一下，讓腳掌平行）
  • 膝蓋微彎，不打直——輕輕地從打直狀態放掉，不是往內夾

- 如果一放就往內夾，需要練蹲姿：蹲下去用手肘把膝蓋撐開來

  • 收小腹，收尾椎（讓腰椎盡量直，不要前凸）
  • 雙肩自然下垂，大拇指輕貼褲縫

- 腋下有微微的空間，不夾腋下

  • 收下巴，讓頸椎與脊椎可以暢通串連
  • 舌頂上顎（讓任督二脈的氣接通）
  • 頂頭懸（從頭頂到尾椎想像一條線垂放，是直的）

這一層調完，架構就有了一個基本的平衡基礎。

這幾個架構上面的話，就是讓身體的架構是不會是歪斜的，然後是比較正的一個基本準則。

四、站樁第二層：調肌肉

骨架調好之後，開始從腳底往上，一層一層調整肌肉的使用量，讓多餘的力量都盡可能放掉。

重心定位：先重心慢慢往前，再慢慢往後，找到一個讓腳踝和小腿壓力最小的位置，停在那裡。這是整個站樁的重心基準點。

依序往上掃描放鬆：

  • 小腿 → 膝蓋（微微放鬆，有支撐力的同時不過緊）
  • 大腿前側、大腿側面、大腿後側
  • 臀部 → 尾椎 → 腰部 → 背部
  • 前面：小腹 → 胸腔
  • 肩膀 → 兩手臂 → 手指頭
  • 頸部 → 臉部肌肉 → 舌頭

能調的就調，調不動的先留著。「知道它的狀況」就已經有機會。

知道他狀況就有機會，你知道他狀況就有機會變鬆，所以不用硬調。

調完之後，腳底和腳踝的壓力會越來越重，因為身體鬆了，重力都往下堆。這是正常的。快速掃描完頸部和臉部之後，就進入第三層。

五、站樁第三層：調心態（等待而覺知）

這是站樁裡面最微妙的一層，也是最容易被跳過的。阿郎稱它為「降低意識狀態」——不是不清楚，而是保持覺知但不多事。

眼睛：往前看，不抓焦點，像看大螢幕。讓光線自然進來，感知哪邊亮、哪邊暗。等那些資訊越來越清晰，眼前的空間會慢慢變清楚，視野會慢慢開闊。

耳朵：不主動去抓聲音，讓聲音進來。心靜下來之後，你會開始聽到聲音裡更多的層次——比如說話的聲音裡有回音，平時聽不到，心靜了才聽得到。

呼吸：知道自己在呼吸，不刻意變成深呼吸。觀察深淺快慢，觀察身體的其他部位有沒有跟著呼吸膨脹和收縮。

意識不多事，只有保留知道的成分就好。看到緊，能調就調；僵化的先看著它，知道自己在呼吸——那個僵化也一直在變。

這個狀態就是站樁的核心狀態，也是打拳的理想狀態。

六、用意不用力的概念

阿郎說明太極拳「用意不用力」的意思：不是完全沒有力量，而是把多餘的力量先清掉，清到最少的狀態。到了那個最少的狀態，你的念頭一動，身體就跟著動——這就是「用意」。

要達到這個狀態，現階段要做的就是：站樁裡面的第一、二層，把多餘的力量盡量放掉。做到了，打起拳來自然就輕。

太極拳裡面講求著「用意不用力」，所以你要先把多餘的力量先把它清掉，甚至就是等於說念頭只要動一些些，身體就要開始動。

七、學員問答

Q：感覺不到自己在呼吸怎麼辦？ A：先試著躺著，躺著的時候呼吸比較深，比較容易感受到。或是觀察胸腔有沒有微微的鎖著——如果有鎖著，就嘗試讓那個鎖著放掉，空氣才進得去。不需要刻意深呼吸，只要知道自己在呼吸就好。

Q：腳底很脹怎麼辦？ A：試試吸氣的時候，想像把腳底的壓力往上提，最多提到腹部這邊就好，不要往頭部提。吐氣時壓力往下，吸氣時稍微減緩一下。

Q：什麼時候用快、什麼時候用慢？ A：慢是觀察工具，用在練習的時候——你要清楚那個過程。快是整合測試，用在熟悉了之後——讓身體看它能不能串連起來。先由慢入手，後面再用快做整合。

八、回家練習與下次銜接

  • 肩胛骨拉筋（前後各一組），在緊的邊界停著呼吸，放鬆時也要慢。
  • 站樁三層：骨架 → 肌肉（從腳往上掃描）→ 心態（等待覺知）。腳酸了就動一動再繼續，不要硬撐到僵化。
  • 每次站樁結束後，動一動膝蓋和腳踝做轉換。

下次課程繼續加深站樁的練習，並開始將站樁的狀態帶入拳架的動作中。

本堂是這個課程中對站樁介紹最完整的一次，三個層次的架構——骨架、肌肉、心態——給了學員一個清晰的練習地圖。每次站樁都可以用這個地圖來自我檢核，知道自己在哪個層次、需要調整什麼。這也是帶回家最實用的練習架構。
