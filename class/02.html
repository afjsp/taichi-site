第二堂課

2026.02.13
本堂內容

一、本堂核心

這堂帶入第二個元素：呼吸。重點不是學習特定的呼吸技術，而是認識深呼吸與腹式呼吸對身體狀態的不同影響，並學習用「氣往腳底」的觀想把重心往下帶。現代人習慣用腦，氣容易往上積，頭重腳輕。腹式呼吸加上觀想，是把重心從上半身往下走的工具。

二、深呼吸與腹式呼吸的差異

深呼吸把空氣撐進胸腔，注意力在上半身，做完容易頭脹或輕微暈眩，氣往上跑。腹式呼吸則是讓肚子膨脹、收縮，注意力落在腹部，身體感覺比較紮實。阿郎說的呼吸練習，重點是進一步的觀想：吐氣時把氣一路導到腳底出去，讓重心自然往下走，不是強迫呼吸、也不是要控制氣，而是讓注意力落在下方。

你關注哪一邊，它就會跑哪一邊。如果我關注在我的胸腔，氣就會聚集在胸腔，壓力就會大。

三、走路練習

走路時把注意力放在腳底板與地面接觸的感覺。不用配合特定呼吸節奏，只要知道「我的覺知在腳底」就好。走路本來腳跟先著地，暫時不需要刻意改變走法，只是把意識往下帶。走完會發現腳底有哪裡疲勞緊繃——那是好事，你看到了，接著可以熱敷、按摩讓它疏通。

四、小腿拉筋

腳跟放在止滑墊（靠牆角），後腳打直，站起來就已經是最繃的狀態，不需要身體再往前壓。拉筋的順序：先三次吐氣把上半身鬆下來，再留意吸氣讓腿部放鬆，然後從小腿開始往上放鬆——小腿、膝蓋後方、大腿後方、臀部、腰部、背部、肩胛骨、頸部。每一層放鬆讓拉筋往上延伸一段，能延伸到哪裡就到哪裡，不強求。頸部拉筋則是用重心平移（不是用手壓），搭配呼吸觀察肌肉鬆緊變化。

五、回家練習

  • 腹式呼吸：每天幾次，感覺肚子膨脹收縮，吐氣時觀想氣到腳底。
  • 走路覺知：步行時把注意力放在腳底板，不改變走路方式，只是讓覺知往下走。
  • 小腿拉筋：腳跟靠牆，站起來拉緊後搭配呼吸，一層一層往上放鬆，能到哪裡算哪裡。
  • 臂體搖擺（複習上次）：頭垂放，輕輕擺動手臂，鬆背部脊椎兩側肌肉。

一、這堂課真正要練的是什麼

這堂帶入第二個元素：呼吸。課程結構依然先複習上週（鬆緊），確認大家對臂體搖擺與雙人擺手的理解，再進入新主題。

阿郎說的呼吸不是氣功或導氣，而是把「呼吸」作為工具，幫助把注意力和重心往下帶。現代人用腦多，上半身壓力大，氣容易往上積，頭重腳輕。這堂的練習核心是：透過腹式呼吸加上觀想，讓身體的重量自然往下走。

二、上週複習重點

臂體搖擺的兩個關鍵：第一，頭一定要自然垂放，不要抬頭，抬頭力量會被鎖住；第二，高點位置決定要鬆動的區塊——要鬆哪裡，就把高點調到那個高度。手背相對（掌背相對）可以讓肩胛骨翻開，幫助鬆動肩胛骨縫隙的肌肉。做完之後慢慢起來：膝蓋往前放，讓重心頂回來，再慢慢站起。

三、深呼吸 vs 腹式呼吸：親身觀察

阿郎讓大家先做三次深呼吸，觀察上半身的狀態。大家的回饋：肩膀會抬起、背部有緊縮感、注意力集中在吸吐的動作、有人頭脹或微暈。這就是深呼吸讓氣往上走的結果。

接著換腹式呼吸：觀想空氣吸進來後流到腹部，讓肚子膨脹，吐氣時收縮。大家的回饋：覺知的範圍比較低，手掌或腳底會熱，肚子會熱，有人微暈的狀態消失了。

你關注哪一邊，它就會跑哪一邊。如果我關注在我的胸腔，氣就會聚集在胸腔，壓力就會大。

阿郎解釋：深呼吸讓注意力和氣集中在上半身，頭重腳輕；腹式呼吸讓重心往下。我們的目標不是執著在氣的具體走法，而是讓上半身的重量重心可以隨著意念和呼吸往下走。

四、觀想吐氣到腳底：重心往下的工具

進一步的練習是：吐氣時觀想氣一路走到腳底出去。這不是真的要控制氣，只是用觀想幫助把注意力往下帶。

有學員問到「把氣往下拉」與「把氣排出去」的差別。阿郎說：重點不在氣的方向，重點是把上半身的重量重心往下帶。觀想只是方便操作的工具，不要太執著在「氣是上還是下」。

五、走路練習：注意力放在腳底

阿郎讓大家在教室裡走動，只做一件事：把注意力放在腳掌與地面接觸的感覺。走快走慢不限，也不要求特定腳法，就是一般走路。

重點是：走路時注意力往下，氣就自然往下跑。如果覺得呼吸的觀想干擾太多，就把呼吸拿掉，單純留意腳底就好。走完後發現腳底某些地方疲勞或緊繃——這是好現象，表示你看到了。接下來的做法就是疏通：熱敷、按摩、或用鬆緊練習讓它放鬆。

六、小腿拉筋：體重拉筋的操作邏輯

這堂介紹了太極拳裡用「體重」而非「肌肉力量」做拉筋的邏輯：不是用肌肉把筋拉開，而是用身體本身的重量，讓筋自然延展。

操作方式（腳跟靠牆角）

止滑墊放在靠牆角位置，一腳腳跟放上去，後腳打直（不要身體往前壓），光是站起來就應該已經是最緊的狀態。如果沒有拉到，代表距離太遠，要往前調。

拉筋的順序分三層：

  1. 先三次吐氣：上半身放鬆往下走（目標是讓重量往下）
  2. 吸氣讓腿部放鬆往上走（感受下半身從腳往上鬆）
  3. 從小腿開始，一層一層往上觀想放鬆：小腿 → 膝蓋後方 → 大腿後方 → 臀部 → 腰部 → 背部 → 肩胛骨 → 頸部

每一層放鬆，拉筋就往上延伸一段。遇到某一層放不過去，就停在那裡，觀察呼吸時那裡的肌肉有沒有鬆緊變化——吸氣微微緊，吐氣微微鬆，跟著這個節奏，它自然會慢慢放開。

有學員做到小腿到臀部都可以，腰部還放不過去，這是架構問題，需要先調整骨架，不是靠拉筋就能完全解決。

腳踝架構要注意：腳不要外翻或交叉，腳掌要與牆面平行。架構不對拉筋沒有效果，甚至會拉到不對的地方。

七、頸部拉筋：重心移動而非肌肉拉扯

一種頸部與肩膀的拉筋方式：身體保持直的，重心平移到一側（用臀部和下半身帶動，不是上半身側傾），頭輕輕往斜上方傾，懸空那側的手自然垂放，不施力。這樣頭的重量和手的重量加起來，自然就會把頸部側面拉開。

關鍵：移動的是重心（下半身主導），不是肌肉在強拉。搭配呼吸觀察那塊肌肉的鬆緊變化，每次吐氣它微微鬆，慢慢就能延展更多。回正時也用重心：支撐腳放鬆，讓重心自然回正，再慢慢站起。

八、學員問答重點

有學員做小腿拉筋時腳底痛而非小腿痠，這是腳部架構問題（扁平足或外翻），可先改用毛巾勾在小腿做拉筋，或用階梯懸腳跟的方式。

有學員建議泡醋水（稀釋白醋）幫助筋骨柔軟，阿郎說這是可以的，不過泡腳機要看說明，有些不能加醋。

架構問題的基本原則：骨架先調（找整脊），同時練習肌肉的使用方式，兩件事要同步進行，不能只調骨架不練肌肉，否則很快又歪回去。

九、回家練習與下次銜接

  • 腹式呼吸：每天幾次，感覺肚子膨脹收縮，吐氣觀想氣到腳底。
  • 走路覺知：步行時把注意力放在腳底板，不用改走法，只讓覺知往下走。
  • 小腿拉筋：腳跟靠牆，後腳打直，搭配呼吸，一層一層從小腿往上放鬆。
  • 臂體搖擺（複習）：頭垂放，輕輕擺動，鬆背部兩側。

下次上課將繼續頸部拉筋練習，也會慢慢帶入更深層的肩胛骨鬆動。
