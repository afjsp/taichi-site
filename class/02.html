🧘‍♂️ 課堂要點紀錄：身心放鬆與結構調整

一、 核心觀念：尋找問題的源頭

  • 樹木的比喻 練功與修行的過程相似，若只修剪葉子（表面習慣），它還會長出來。必須往回找，從細枝、樹幹到樹根（源頭），處理掉源頭，身體的放鬆與平衡才容易達成。
  • 先調架構，後調肌肉 當身體有傷或位置不正時，必須先讓骨架（結構）歸位，再練習新的肌肉使用方式。若結構歪斜，練肌肉只會讓身體更失衡。
  • 以緊求鬆 透過極致的延展或增加強度讓身體變「緊」，在放下的那一刻，感知什麼是「相對的鬆」，進而提升對鬆緊的覺察力。

二、 呼吸的辨識與運用

音檔中詳細比較了三種呼吸方式對身體的影響：

  • 胸式呼吸（深呼吸）： 氣容易往上衝，導致肩膀抬起、後腦發脹或頭暈。
  • 腹式呼吸： 觀想空氣流到腹部，吸氣時撐大、吐氣時收縮，能讓覺知較容易集中在軀幹。
  • 意念導引（吐氣到腳底）： 為了改善「頭重腳輕」的狀態，吐氣時觀想氣一路走到腳底，能幫助上半身放鬆，將重心下移。

三、 重點實作練習

1. 背部與脊椎兩側放鬆

  • 動作： 頭部自然垂放（下巴微收，不要抬頭），雙手下垂，利用手部的重力帶動身體輕微彈動。
  • 要點： 不同高度能調到不同位置。若要鬆肩胛骨，高度要稍高；若蹲太低，力量會跑到腰部。

2. 小腿拉筋（結合呼吸與放鬆）

  • 工具： 使用止滑墊墊在牆角，腳跟踩地，腳掌貼牆站起。
  • 進階引導：
  • 結構： 腳掌必須與牆面垂直，不可歪斜，否則會造成腳踝走位。
  • 由上往下放鬆： 搭配吐氣，將上半身的壓力卸掉。
  • 由下往上延展： 依序觀想並放鬆小腿、膝蓋後方、大腿後側、臀部、腰部、背部、肩胛骨，最後到頸部。
  • 替代方案： 若因腳掌結構（如扁平足或舊傷）無法使用止滑墊，可改用樓梯台階（懸空腳跟）或毛巾拉筋。

3. 頸部與肩部拉筋

  • 動作： 保持脊椎挺直，移動重心（非上半身出力）向一側傾斜，頭部微偏，手輕放對側頭部增加重量，另一手垂放。
  • 要點： 利用頭部與手的自然重量產生拉力。在緊繃感中觀察呼吸，留意吐氣時肌肉微幅變鬆的狀態，並在那種鬆的感覺中持續延展。

4. 覺知走路

  • 練習： 走路時將注意力單純放在腳掌與地面的接觸感。
  • 目的： 將專注力放低。如果搭配呼吸太過干擾，則先放掉呼吸，只專注於腳底，氣自然會隨意念往下走。

四、 課後建議與保養

  • 泡醋： 若筋骨過於僵硬或有舊傷不通順，老師建議可以用不稀釋的白醋泡腳（可加溫），有助於軟化筋骨與排毒。
  • 觀察鬆的變化： 練習時不需刻意盯著緊處，而是知道哪裡緊，然後觀察它在呼吸過程中產生的細微變化，讓它像「化開」一樣自然鬆掉。

課程要點（詳細版）

一、課程背景與複習

  • 複習第一堂鬆緊概念，確認學員對「鬆」的相對定義有初步體驗
  • 本堂導入第二元素「呼吸」——太極拳以呼吸驅動鬆緊
「你把氣往哪裡送，身體的重心就往哪裡走。意念跟呼吸是綁在一起的。」

二、深呼吸 vs 腹式呼吸的區別

【深呼吸（常見錯誤）】

意念放在胸腔，氣往上跑 → 頭重腳輕、容易頭暈

胸腔用力擴張，肩膀容易聳起 → 上半身更緊

【腹式呼吸（太極拳要求）】

意念放在腹部，腹部隨吸氣膨脹、吐氣收縮

氣往下走 → 重心跟著往下 → 腳底接地感加強

吐氣時，可以觀想氣從腳底出去，練習「上半身放下來」的感覺

  • Q：要一直用腹式呼吸嗎？感覺好累。

A：不是一直刻意做，而是讓它慢慢成為習慣。一開始先在練習時做，習慣了就會自然帶入日常。

三、意念與氣的關係

  • 意念往哪裡，氣（注意力帶來的身體感覺）就往哪裡聚集
  • 日常可以觀想腳底：睡前觀想腳底呼吸 → 有助入睡、幫助穩定重心
  • 太極拳的意念不是「想像」，而是真實的身體感知練習

【呼吸與鬆緊的對應關係】

吸氣 → 身體壓力微增（相對緊）

吐氣 → 身體壓力微降（相對鬆）

→ 把注意力放在某個緊繃的部位，配合吐氣，那裡就會慢慢鬆開

「你不需要很厲害的技術。你只要吐氣的時候，把注意力輕輕放在那個地方，就夠了。」

四、靠牆小腿拉筋（詳細步驟）

【準備動作】

腳跟放上止滑墊靠近牆角（或直接靠牆），後腳站直

光站上去時就是最緊繃的狀態——這就是起點

【前三個呼吸：吐氣放下來】

① 不要用力拉，讓身體的重量自然往下沉

② 吐氣時，從上往下：肩頸 → 胸腔 → 腹部，依序放鬆

③ 觀察吐氣後，拉伸的感覺有沒有稍微多一點點

【後三個呼吸：吸氣帶起來】

① 吸氣時，意念從下往上：小腿 → 膝蓋 → 大腿 → 臀部 → 腰 → 背 → 頸部

② 每個部位吸氣帶過去時，感受它稍微放鬆一點

③ 這個方向是把腳部的壓力減輕——讓氣從地面帶上來

【觀察點】

你能拉到哪裡（小腿／膝蓋後側／大腿後側），代表你目前的放鬆程度

不要比較誰拉得深，每次比自己上次多一點點即可

【注意事項】

腳踝不要外翻，兩腳要與牆面平行

舊傷者可以改用直背前傾方式延展脊椎，不用強拉

頭暈時立刻停止，走動一下讓氣歸位

五、脊椎直立的意義

  • 下巴收進來、髖骨前緣與下巴對齊，讓頸椎保持直線

→ 現代人常見「頭往前伸」姿勢（電腦病），頸椎斜掉讓氣容易卡在上頭

  • 收下巴時脊椎拉直，配合吐氣 → 上半身壓力更容易往下走
「你以為你在看螢幕，但其實你的頭早就往前一截了。把下巴輕輕收回來，那一截就還給脊椎了。」

六、走路中的呼吸練習

  • 走路時，把注意力放在腳掌與地面接觸的感覺
  • 配合吐氣，觀想氣從腳底出去（不用很用力想，輕輕帶過就好）
  • 腳底外緣如果有浮腫感，可用大彈珠踩（調整足弓），每次踩兩顆

七、問答重點節錄

  • Q：我試著把氣送到腳底，但感覺沒有什麼反應。

A：正常，一開始就是這樣。你只要有在做這件事，身體就在慢慢學習。不用急著有感覺，那個感覺會自己來。

  • Q：腹式呼吸要肚子凸出來嗎？感覺怪怪的。

A：不是刻意讓肚子凸，而是讓腹部自然地接收氣，不去阻擋它。你試著吸氣時別讓胸腔動，看腹部有沒有自然膨一點。

八、課後練習建議

  • 睡前：躺著，觀想腳底呼吸 3-5 次（助眠練習）
  • 走路時：每走幾步就把注意力放回腳掌，感受接地感
  • 靠牆小腿拉筋每天早晚各一次，每次 6 個呼吸
  • 注意：不要一直「深呼吸」，讓呼吸自然，只是把注意力輕輕放到腹部
