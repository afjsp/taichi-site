第八堂課

2026.04.10
本堂內容

一、本堂核心

這是八堂基礎課的最後一堂，把站樁、步伐兩個主題深化，並在課程末尾帶出起式的口訣——讓大家知道下一期要打的東西是什麼樣子。阿郎在最後說的一句話可以作為這八堂課的總結：只要帶著放鬆移動的概念在生活裡，你已經在練太極拳了，不需要等到動作練熟了才算。

二、站樁：三層架構再複習

站樁的三個層次：第一層架構（雙腳平行與肩同寬、膝蓋微彎、收小腹收尾椎、肩膀自然下垂、收下巴頂頭懸）；第二層逐層放鬆肌肉（從腳踝到頭皮）；第三層耳眼等待狀態（讓聲音光線自然進來，不主動抓）。打拳就是帶著這三層狀態在動。

三、步伐：虛步與移動的放鬆邏輯

走路的移動方式根本上是靠放鬆，而不是靠用力推進。往前走不是用後腳推自己，而是前胯放鬆、重心滑過去；後腳要越來越鬆，才能順勢輕輕提起。前腳落地時要輕——像貓步，先確認地面沒有陷阱，確認穩了才把重心鬆過去。這就是太極拳的虛步：腳放下是輕的，確認了再讓重心進去。

你帶著這個概念在生活裡頭，事實上你不用去練習到原本的動作，你都已經開始在練習太極拳。

四、起式初探

課末阿郎示範了起式口訣，作為下一期的「緣起」：

起式：收肘、垂手、提腕、起手、伸指、屈臂、豎掌、伸手、沉手、合手

動作本身沒有想像中複雜，口訣只是路徑提示。動作裡要練的不是外型，而是每一個步驟如何用身體帶動、肌肉如何均勻放鬆。

五、八堂課的總結

這八堂課的核心概念：均勻、放鬆、流動。蹲姿和坐姿是練架構；拉筋是調整肌肉（第二層）；走路和站樁是讓這些狀態進入日常。下一期從洋式太極拳第一段開始，架構只是第一層——大家要帶著這八堂練到的第二層和第三層繼續走。

一、這堂課真正要練的是什麼

這是八堂基礎課的最後一堂，把前幾堂積累的觀念做最後整合，並在課末帶出下一期的起點——起式的口訣。

阿郎在這堂說了這八堂課的終極目標：

你帶著這個概念在生活裡頭，事實上你不用去練習到原本的動作，你都已經開始在練習太極拳。

這八堂課沒有教「一套拳」，而是讓大家認識身體、認識放鬆的語言。帶著這個語言，往後打任何動作都是在練同一件事——均勻的鬆。

二、站樁三層次完整複習

站樁是打拳前的準備狀態，也是打拳時整個過程需要維持的狀態。這堂再次完整走了三個層次。

第一層：架構

雙腳平行站立與肩同寬（腳尖微往內，不外八）。膝蓋微彎（不打直），讓膝蓋在腳掌的正上方，不往內夾也不過度往外翻。收小腹，讓尾椎往下，腰椎的弧度減少，接近一直線。肩膀自然下垂，大拇指側輕貼褲縫，腋下微微打開。收下巴讓頸椎對齊脊椎，舌頂上顎，頂頭懸讓整條脊椎往上延伸。

這個架構，如果膝蓋容易往內夾，就要加練「蹲姿撐開膝蓋」——雙手肘撐在膝蓋內側，讓膝蓋往外的方向找到正確的位置。不要在架構偏掉的情況下繼續練，因為錯誤的架構會從腳踝一路影響到脊椎。

第二層：肌肉調整

架構確定後，從腳踝開始往上逐層放鬆：找到腳掌均勻接觸地面的位置（往前讓腳趾頭浮起、往後讓腳跟浮起，中間是均勻點），然後讓腳踝、小腿、膝蓋、大腿、臀部、背部、橫膈膜、兩臂、頸部、臉部、舌頭、頭皮一層一層放掉。不是整個鬆垮，而是在維持架構的前提下讓肌肉的張力降到「剛好夠用」。

透過呼吸觀察：某個部位如果感覺緊，把注意力放在那裡，觀察吐氣時它有沒有微微鬆動。能隨呼吸起伏的肌肉就是活的，不會繼續僵化。

第三層：整體狀態

主意識稍微降低，讓耳朵和眼睛進入「等待」的狀態：聲音自然進來，不急著貼標籤；光線自然進來，眼睛不主動掃視。知道自己在呼吸，感受到每次呼吸身體都在微微變化。這個「知道但不抓」的狀態就是第三層。

站樁不是要撐，是要維持在能調整的範圍裡。感覺到僵了，用意念調；意念已經調不了，就起身動一動再繼續。

三、步伐：虛步與放鬆移動

這堂對走路的步法說得更清楚了。

移動的本質是放鬆，不是用力推

往前走，不是後腳推前行，而是前胯放鬆，重心自然往前滑。後腳也不是一直抓著地，而是隨著重心前移越來越鬆，最後輕輕提起。用力推的結果是上下起伏大，重心不穩；用放鬆移動的結果是重心維持同一高度「平移」過去，上半身幾乎不動。

貓步（虛步）的邏輯

前腳落地是輕的——這叫做「虛步」。不是說腳落地不要用力，而是這隻腳「還不確定地面的狀況」，所以先輕輕放上去，確認穩了再把重心鬆過去。這樣即使踩到陷阱，也可以收回來。用力踩下去就直接掉進去了。

確定完之後，我們重心才慢慢的鬆上去。這太極拳在走步的時候的這個毛步（貓步）的時候，是確定完了再讓重心過去。

重心的轉換

支撐腳要保持均勻接觸地面，但不能「壓死」——一定要還有可以放鬆的元素，這樣對側腳才能輕輕地動。具體做法：支撐腳掌均勻踩穩，然後稍微有一個放鬆的觸發（翹翹板），前腳就能輕輕起來。

一開始可以走得很慢，感受每一步的重心轉換。熟了之後速度自然起來，不需要刻意控制。

四、身體不均勻的調整方式

有學員問到左右受力不均（一邊受傷導致習慣偏側）怎麼調。阿郎說：

太極拳的調整方式不是刻意去「拉正」偏掉的一側，而是找出緊繃的地方，把緊的部分練鬆。兩邊同時練習放鬆，放鬆的程度會越來越接近，自然就趨向平衡。你只有在做一件事：放鬆。

你不用刻意在調，你只要找到說我這一邊、這一邊的話他是比較緊的，我就單純的把緊的東西放掉。但是這個狀態的話他就是達到某一個左右平衡。

五、起式口訣——下一期的緣起

課末阿郎帶出了起式的口訣，作為下一期正式學拳的「開頭」：

無極（站樁準備狀態）→ 太極拳起式：

收肘、垂手、提腕、起手、伸指、屈臂、豎掌、伸手、沉手、合手

動作本身並不複雜。口訣是路徑的提示，不是要背的咒語。重要的是：每一個步驟裡，你是「手在動」還是「身體帶動手」？這個差異，就是第一層（太極操）和第二層（太極拳）的區別。

⚠ 拳譜核對說明：本堂逐字稿中起式口訣有 Whisper 誤聽（伸直→伸指、曲背→屈臂、舒展→豎掌、承手→沉手），詞庫系統無法自動攔截，已在逐字稿中手動修正，summary 使用正確名稱。

六、八堂課的總結與下一期

阿郎給了這八堂課一個清楚的定位：

  • 蹲姿 / 坐姿 = 練架構（第一層）
  • 拉筋 / 呼吸放鬆 = 調整肌肉均勻（第二層）
  • 站樁第三層 / 走路覺察 = 整體狀態和意識（第三層）

下一期開始正式學洋式太極拳第一段。每個動作一開始都是在講架構（第一層），但大家的任務是帶著第二層和第三層進去——讓動作從「太極操」變成「太極拳」。

學會洋式太極拳這件事情是附屬的，他是來調整說自己的身體可以用這種方式來練習到一定的均勻一定的鬆。

七、回家練習（這期結語）

  • 站樁：三層依序進入，僵了就動，不硬撐。
  • 走路：前腳輕放確認再鬆重心，重心維持同一高度平移。
  • 起式口訣先熟悉：收肘、垂手、提腕、起手、伸指、屈臂、豎掌、伸手、沉手、合手。
  • 觀念帶進生活：任何時候用放鬆的方式移動，就是在練太極拳。
