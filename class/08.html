一、 站樁的三個層次 (Three Levels of Standing Meditation)

站樁是進入太極拳的基礎，可分為由外而內的階段：

  • 第一層：架構（身體支架）
  • 足部與膝蓋：雙腳平行站立與肩同寬，不外八。膝蓋微彎並對準腳掌上方，不內夾也不過度外撐，確保腳掌與地面均勻接觸,,。
  • 軀幹調整：收肚子與收尾閭以修正腰椎弧度；雙肩自然下垂，大拇指輕貼褲縫（確保腋下暢通）,。
  • 頭頸與口腔：收下巴使頸椎與脊椎連成一線（視線仍維持平視）；舌抵上顎（門牙後方）；頂頭懸感受脊椎自然垂掛,,。
  • 目標：達到「放鬆但不鬆軟」的狀態，維持正確架構。
  • 第二層：肌肉（放鬆與流動）
  • 均勻受力測試：微幅前後移動重心，找出腳踝與小腿受力最小、最穩定的中心點。
  • 循序放鬆：由下而上觀察小腿、膝蓋周圍、大腿、背部與肩頸。練習用最少的力量支撐身體,。
  • 配合呼吸：若覺察某處緊繃，將注意力留在該處並隨呼吸起伏，等待肌肉自然鬆開,。
  • 目標：追求肌肉的平衡與流動性，而非整個人鬆垮。
  • 第三層：意念與環境（整體性互動）
  • 等待的狀態：降低主觀意識，對環境的光線、聲音保持「知道」但不貼標籤、不刻意抓取的開放狀態,。
  • 目標：將架構、肌肉與環境融合，達成意念帶動身體的輕靈感。

二、 步法練習與重心移動 (Walking & Shifting Weight)

  • 重心的平穩移動
  • 練習在移動過程中維持重心高度不變（不忽高忽低），避免像跳躍般的起伏。
  • 放鬆帶動移動：先放鬆前胯與膝蓋，再放鬆後胯與臀部，利用肌肉的鬆開產生滑動感，而非用大腿肌肉生硬地「推」,。
  • 貓步（虛步）的練習步驟
  • 確認支撐：重心移至一腳，確保該腳腳掌受力均勻且不僵硬,。
  • 輕靈提放：提膝時小腿與腳踝需完全放鬆，不帶支撐力,。
  • 虛觸地面：前腳伸出時像試探陷阱一樣輕觸地面，確認穩固後才移轉重心。若有狀況，應能隨時收回而不失去平衡,。

--------------------------------------------------------------------------------

三、 練習要領與建議

  • 關於鞋具與足部架構
  • 建議選擇薄底、柔軟（如皮質）且能讓腳掌完全貼地的鞋子，以增加對地面的感官清楚度,。
  • 避免穿著過厚或回彈力太強的運動鞋，以免腳趾不自覺抓地（代償作用）導致緊繃或受傷,。
  • 透過「踩珠子」等練習強化足弓，提升腳踏實地的飽滿支撐感,。
  • 日常生活的覺察
  • 觀念重於招式：不一定要練整套動作，在走路、拿東西或坐著時，留意是否使用了過多蠻力，練習用均勻、放鬆的方式運作身體。
  • 少量多次：練習應是有感且愉快的，像喝水一樣輕鬆。若感到疲勞或意念太重，應適度收掉、隨意動一動,,。
  • 觀察即是修正：發現緊繃時，單純地把緊的地方調鬆，左右自然會趨於平衡。

二、站樁三層次——完整詳述

【第一層：架構（身體的支架）】

① 雙腳平行與肩同寬（腳趾朝前，非外八；可腳跟稍往外翻再放下調整）

② 膝蓋微彎，且要在腳掌正上方——不能往內夾，也不能往外過翻

→ 膝蓋內夾會影響腳踝→臀骨→脊椎，引發連鎖偏頗

③ 收肚子，讓腰椎不過度凹陷；順帶收尾閭，輕輕往下放

④ 雙肩自然下垂，大拇指輕貼褲縫（腋下微開，不夾緊）

⑤ 收下巴，讓頸椎與脊椎串聯成直線（視線仍往前看，不是看地板）

⑥ 舌頂上顎（門牙後方輕放），頂頭旋（頭頂往上延伸）

「架構在的話身體就不會鬆軟。我們要的是有架構的鬆，不是垮掉的鬆。」

【第二層：肌肉調整（暢通的平衡）】

→ 在架構不崩的前提下，由下往上逐一掃描各部位肌肉：

腳踝、小腿 → 膝蓋周圍 → 大腿（前後左右均勻）

→ 腹部（稍微收住）、背部（可放鬆）、肩膀 → 兩手臂

→ 頸部（收下巴但放掉多餘的力）→ 臉部、眼周、耳朵、頭皮

→ 觀察方法：哪裡不能隨呼吸起伏，就在那裡多留專注力

→ 隨著呼吸能微微起伏，那裡就開始鬆了

→ 臉部肌肉不知道怎麼放鬆：回到覺察呼吸——身體整體舒張收縮，臉也會跟著改變

「肌肉調整不是整個鬆垮，而是要讓他稍微能流動。流動就不僵，不僵就鬆。」

【第三層：與環境互動（意識的均衡）】

→ 帶著等待的狀態感知聲音：不主動貼標籤，只是知道有聲音

→ 慢慢心靜下來，越來越清楚有什麼、有多少種聲音

→ 眼睛微微打開，讓光線自然進來，不聚焦看東西

→ 主意識稍微降低，但仍保持對環境的知道（不是昏昏欲睡）

→ 呼吸自然，知道自己在呼吸就好，不要控制

「你的意識用力的部分放掉，但你還是保持著知道——訊息會進來的那種知道。」

三、站樁與打拳的三層次對應

【核心洞見】站樁的三層次就是打拳的三層次——完全相同的框架

第一層（架構）：教動作時討論的就是這層——手在哪裡、身體怎麼轉

第二層（肌肉）：練習時老師提示放鬆調整——這是第二層

第三層（意念）：打起來的感受，意念是否輕盈帶動身體——這是第三層

「我們在教動作的時候，都只有在討論架構而已。後面還有兩個層次可以慢慢走。」

【手帶 vs. 身體帶的差異】

手起來，可以只是手在動；也可以是身體帶動，手只是跟著

→ 看起來一樣，感覺和效果完全不同

→ 太極拳要練的是「用身體帶動」——這是第二層往第三層走的路

四、用放鬆觸發移動——走路（貓步）詳解

【移動的核心邏輯：不是推，是鬆】

一般人移動 = 肌肉推出去（前腳用力踏）

太極拳移動 = 前胯放鬆 → 後胯放鬆 → 身體自然滑動

→ 前面放了，後面還在煞車 → 後面也要放，才真正流動

「就好像車子要往前衝，但後面有個煞車把你煞住。你把後面放掉，就開始有東西鬆動了。」

【重心平移——走路的目標狀態】

重心維持在同一高度，不要上下起伏

→ 起伏大 = 走路像跳舞或像兔子跑；起伏小 = 像滑行，像溜冰

→ 起因是肌肉放鬆到均勻，「果」自然就是平移

【貓步步驟拆解】

① 重心確認在支撐腳，支撐腳腳掌均勻接觸地面

② 移步腳輕輕放出去（虛步）：腳落地是輕的，重心還在後面

③ 確認前腳落點無誤後，才慢慢把重心鬆過來（不是一腳踩死）

④ 支撐腳後腳越來越鬆，能提起時才提起，不要硬抬

⑤ 後腳膝蓋往前的概念帶起腳，而非腳踝用力拉起

「前面有陷阱，腳一碰，它掉下去了，但你重心還在後面——你就可以收回來。這就是虛步的作用。」

【常見問題診斷】

走路歪斜 → 某個平衡機制在出力，找出來放鬆它

後腳提不起來 → 重心不夠前 / 腳跟沒鬆 / 前腳也崩了

腳步聲重（啪扎）→ 鬆得不夠，腳放下去有壓力

步距太大 → 支撐不夠，被迫用腳踝撐；先縮小步距

五、身體不均勻的調整方式

【太極拳的調整邏輯】不是強制壓回，而是只做一件事：把緊的地方調鬆

假設右肩比左肩高（緊）→ 不是把右肩往下壓

→ 只是讓右肩慢慢放鬆，同時兩邊都練習放鬆

→ 兩邊的鬆慢慢靠近，自然達到平衡

「你根本沒有刻意在調。你只找到說我這一邊比較緊，把緊的放掉。這個狀態就達到了左右平衡。」

【為什麼這樣對長短腳也有效？】

長短腳往往來自某側肌肉長期緊繃代償

→ 把緊的調鬆 = 消除代償 = 結構逐步歸正

→ 持續均勻受力練習，改善效果更快顯現

六、鞋子選擇與足弓的關係

【為什麼要穿平底鞋？】

太極拳需要清楚感知腳掌跟地面的接觸

厚底、氣墊鞋 → 地面感受模糊 → 腳趾不自覺勾起抓地

腳趾抓地 → 臀部出力代償 → 長期下來屁股或腰會痛

「你在打拳的時候就是會整個用不同的地方去代償。那我就會用那個屁股的力量去平衡。」

【腳踏實地的感覺是什麼？】

腳的架構對了，踏上去的感覺是「飽滿的」

→ 受力均勻、有踏實感、不搖晃——這就是腳踏實地

→ 架構沒對時，感受到的只是「接觸」，不是「支撐」

【換鞋建議】

先用踩珠子（大彈珠）調整足弓，讓足弓有弧度有支撐力

足弓調整後再慢慢換平底鞋，不要躁進

不同運動配不同鞋：羽球鞋防側向摩擦、太極需平底——配合情境

七、前行課總結——觀念比動作更重要

【八堂課學了什麼？】

三個元素（鬆緊、呼吸、動態平衡）＋ 架構（坐姿、蹲姿、站樁、走路）

→ 這些是「讓你認識自己身體」的方法，不只是功夫練習

【為什麼觀念比動作更重要？】

帶著「用放鬆的方式移動」「留意整體架構」的觀念在生活中行動

→ 你就已經在練太極拳了——不限定在拳架中運行

「你不用去練習拳架，世可以在練習太極拳。因為核心的概念你是清楚的。」

【蹲姿、坐姿、拉筋的目的回顧】

蹲姿、坐姿 = 校正架構（第一層）

拉筋 = 調整肌肉（第二層）

→ 所有練習都朝向「讓身體越來越鬆、越來越均勻」這個核心目標

【一招練透的力量】

不用急著把整套拳學完，先把一招練得透了

→ 那一招就會幫你點開其他招式——這是太極拳傳遞的方式

八、楊式太極起式啟示（課末加碼）

【觀無極】進入站樁狀態，走完三個層次，清楚自己的狀態

→ 帶著這個狀態就叫做「觀無極」，然後再開始打拳

【起式口訣】

觀無極 → 起式 → 收軸垂手 → 提腕起手 → 伸直曲背

→ 舒展伸手 → 承手合手

「口訣拿掉好像很簡單對不對？就這樣而已。別被口訣唬住了，其實動作就是這樣。」

九、問答重點節錄

  • Q：膝蓋一直內夾，要怎麼調？

A：練蹲姿修正——蹲下來手肘撐在膝蓋內側往外撐。腳踝不健康時先練上半身肌肉部分。

  • Q：穿鞋子比較穩，光腳或平底鞋反而不穩，怎麼辦？

A：穿輔助鞋是暫時穩定，但讓你對地面清楚度降低。目標是調整足弓，讓腳自己能支撐。先踩珠子調足弓，再慢慢換鞋。

  • Q：已受傷的腳，可以靠練習讓它回正嗎？

A：會有幫助，但有邊界。練習方向放在均勻受力，讓緊繃的地方慢慢放鬆，結構就會往正的方向走。

  • Q：可以密集學完108式嗎？

A：三個月可以學完，但要密集練習。更好的方式是先把一招練透，它會幫你點開其他招式。

十、課後練習建議

  • 站樁每天一次：從三分鐘開始，走完三層次為準（不是硬撐時間）
  • 走路練貓步：重心保持等高，每一步輕放確認後才鬆重心過來
  • 問自己：「這個動作可以用更少、更均勻的力量做嗎？」
  • 腿部緊繃後要放鬆：按摩、捶打或慢走各 1-2 分鐘
  • 先選一招練到有感覺，再換下一招——不要一直換
