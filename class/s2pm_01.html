第一堂課

2026.05.07
本堂內容

2026-05-07｜主題：脚底架構、重心轉移，與三個腦袋的關係

一、本堂核心

本堂從脚底開始，讓身體感受什麼叫「均勻站好」。太極拳的移動不是靠推力，而是靠放鬆——放鬆哪邊，重心往哪邊流。動作要慢，因為慢才能看到肌肉的細小變化，身體才有機會自動調整。阿郎也介紹了一個重要概念：我們有三個腦袋，太極拳是訓練本能腦和情緒腦，讓身體記憶，而不只是用理性腦去背動作。

二、脚底架構與均勻受力

脚掌骨架：腳跟一個支撐點，前掌那一排骨頭是面。脚掌上的肉是感測器，讓它均勻貼地。這個練習本身就是一種感覺練習——你在感受壓力如何分布、如何均勻化。感受方式：很緩慢地把重心往前再往後，觀察每個區域的壓力變化。發現壓力不均，就試著調整，讓均勻的狀態出現。

我們現在大家現在都可以讓腳掌直接跟地面接觸，所以你可以感受一下你現在跟地面接觸的時候，腳掌的受力它是均勻的嗎？

三、用放鬆做重心轉移

往左移：把左腿大腿前側、臀部下緣、膝蓋放鬆一些些，右側髖關節也放掉，重心就流過去了。不是右腳推。往前往後同樣道理。

呼吸調整的技巧：吐氣時觀察「往下放」的感覺，讓頸肩胸背跟著吐氣往下鬆。如果腿腳壓力過大，改關注吸氣，感受能量從腳底往上提的感覺。

四、拳架複習：起勢

收肘 → 垂手 → 提腕 → 起手 → 伸直 → 屈背疏掌 → 伸手 → 成手 → 合手

重點：伸手出去時搭配膝蓋放鬆，手就不用那麼用力。肩膀安靜不聳，合手像抱一個球在小腹前。

五、回家練習

  • 脚底練習：站好後慢慢讓重心往前移，觀察哪條肌肉先動。越慢越清楚。
  • 呼吸調整：每天三次深呼吸，觀察吐氣時身體往下放的感覺，讓上半身跟著鬆。
  • 起勢練習：配合呼吸，肩膀不動，找到起手卡點的高度，每天練三遍。

本堂的核心提醒是：所有的移動都從放鬆開始，不從用力開始。脚底站好是基礎，重心轉移是練習，起勢是把這兩件事結合起來的載體。慢是工具，讓你能看到身體的變化；看到了，身體才有機會自己去調整。每天五分鐘，比偶爾一小時更有效。

帶走這堂課的一件事：放鬆帶動，而不是用力推。

2026-05-07｜主題：脚底架構、重心轉移，與三個腦袋的關係

一、這堂課真正要練的是什麼

本班第一堂的目標是：讓學員第一次感受到，身體有自己的調整機制，而你要做的是「讓它發生」而不是「叫它做」。課程從脚底開始，因為脚是與地面最直接的接觸點；重心轉移從這裡學起，才知道放鬆移動是什麼感覺。阿郎也特別說明了太極拳的本質：它不是教動作，而是教你怎麼透過慢，看到自己身體的狀態，讓身體自己去做修正。

二、脚底架構：骨架支撐，肉是感測器

脚的骨架有三個支撐點：腳跟，以及前腳掌骨頭那一排（一個面）。這是承重的架構。脚掌上的肉不是用來支撐的，而是感測器——它均勻接地，你才能知道受力是否平衡。

站好之後，試著把鞋脫掉感受腳掌與地面的直接接觸：會不會前面比後面重？左邊比右邊重？外側比內側重？有壓力偏大的地方，就要讓它均勻化。

你可以感受一下你現在跟地面接觸的時候，腳掌的受力它是均勻的嗎？

三、慢的道理：A4紙與磁磚的比喻

阿郎解釋為什麼要慢：一根羽毛落在A4紙上，手指能感覺到；落在磁磚上感覺不到——因為兩者重量差太多。同樣，如果你的肌肉力量用太多，速度太快，你就看不到肌肉細小的變化。要把自己用的力量降低，讓它跟那個細小的變化「比例接近」，才看得到。

慢的目的是讓你觀察到身體在每個動作中的肌肉狀態。看到了，身體才有機會自行調整。這個「讓身體自行調整」是課程八堂一直在用的方法。

四、重心轉移：放鬆帶動，不推

太極移動的方式：要往左，不是右腳推——而是把左腿大腿前側、臀部下緣、膝蓋放鬆一些些，同時右側髖關節也放一點，重心就往左流動了。像放風箏的繩子鬆一點，風箏往那邊飄。

一開始是刻意的，你知道要放哪裡；練到後面，你會抓到「放的感覺」，然後就是靠感覺帶動，不再需要想那麼多。這就是從「第二個元素（身體控制）」升到「第三個元素（氛圍感受）」的過程。

呼吸調整的技巧：

上面壓力太大（例如頭頸肩緊）：觀察吐氣，吐氣時想著往下放，讓頸、肩、胸、背跟著往下鬆。

腿腳壓力過大：改注意吸氣，感受從腳底往上提到丹田的感覺，讓下面的壓力稍微減緩。

吐氣的話是上面往下放，最簡單的是單純的留意吐氣的路徑而已，那這部分的話你會發現到上面變輕了然後下面變重，這是自然的狀態。

五、四個元素與三個腦袋

四個元素：動作 → 身體（肌肉覺察）→ 心理（情緒/氛圍）→ 意念。這四個在練習時同時存在，要練的是讓它們頻率一致。可以先從兩個開始：動作＋身體覺察，這樣就已經是在練專注力了。

三個腦袋的解說：

  • 理性腦（前額葉）：想「我要這樣做」，速度慢。
  • 情緒腦：感受到狀態，比理性腦快。
  • 本能腦：求生存反應，最直接。

太極拳不是用理性腦背動作，而是透過刻意練習，讓身體記憶，讓本能腦和情緒腦接管——就像阿郎探勘時腳下陷入，下意識蹲下再跨過去，那個反應不是「想」出來的，是身體記憶的結果。

慢不是目的，覺察才是。慢只是讓我們能觀察自己身體的工具。

六、觀無極與站樁

站樁分三層：

  • 架構層：雙腳平行與肩同寬，膝蓋微彎，收小腹，肩膀自然下垂，收下巴，舌頂上顎，頂頭懸。
  • 肌肉層：從腳底往上掃描，讓小腿腳踝壓力最小，慢慢放鬆往上。
  • 意念層：眼睛不抓焦點，像等待光線進來；耳朵等聲音；知道自己在呼吸。什麼都不做，但保持覺知。

七、起勢動作

收肘 → 垂手 → 提腕 → 起手 → 伸直 → 屈背疏掌 → 伸手 → 成手 → 合手

重點：伸手出去的時候搭配膝蓋微微放鬆，手就不需要那麼多力量。合手像抱大球，不要太往前，保持在腰的高度。

八、學員問答

Q：高血壓的人適合太極拳嗎？ A：非常適合。把意念往腳底放，上面的壓力自然往下走。深呼吸觀察吐氣路徑，讓頸肩胸往下鬆，頭頸的壓力就會下降。

Q：如何讓太極拳的反應進入日常生活？ A：先刻意練，讓身體記住那個「安全的動作模式」。記住了之後，身體遇到類似狀況就會自動反應，不需要再想。

九、回家練習與下次銜接

  • 脚底均勻受力，走路時留意。
  • 重心轉移：用放鬆帶動，左右、前後都練。
  • 呼吸調整：每天三次深呼吸，觀察吐氣往下放的感覺。
  • 起勢複習，每天三遍，肩膀不聳，伸手搭配膝蓋放鬆。

下次課程繼續深化觀無極的三層結構，並加入更多拳架動作。

補充：太極拳在生活中的應用

阿郎強調，太極拳真正有價值的地方，不是在打拳的那幾十分鐘，而是讓這個「覺察的習慣」滲透到日常生活的每一個動作裡。你走路、拿杯子、坐下站起——這些每天重複幾百次的動作，都可以成為練習的機會。

練習的方式很簡單：把速度放慢一點，觀察你在做這個動作的時候，哪裡用了多餘的力量。找到了，試著少用一點。用的少了，你才能看到更細的變化。這個循環就是太極拳的核心練習邏輯——不是動作有多特別，而是你在做任何動作的時候，你是不是有在「看著」它。

所以從這堂課開始，試著把今天學到的「放鬆帶動」的感覺帶到日常生活裡。走路的時候試試看，拿東西的時候試試看。你會發現，原來我們日常的很多動作都是在「硬撐」，而不是「放鬆流動」的。

太極拳的根本在於：每個動作都是一個觀察自己的機會。你在做這個動作的時候，是否清楚地知道哪裡用了力、哪裡可以更鬆？這個「清楚地知道」就是覺察力的積累。隨著你對自己身體的了解越來越深，動作就會越來越輕、越來越流動，不是因為你「更努力」，而是因為你「更清楚了」。這就是阿郎說「動作只是工具」的意思——工具是服務於覺察的，不是目的本身。

今天學到的最重要的一件事，就是「放鬆帶動」。不是不用力，而是用最少的力，讓最多的地方流動起來。這個概念帶走了，練習就永遠有方向。
