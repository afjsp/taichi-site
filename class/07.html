第七堂課

2026.04.03
本堂內容

一、 坐姿與起身的深化：溜滑梯原理

  • 學習坐姿的目的：是為了在靜態中體認「上半身放鬆」的狀態，並將這種狀態帶入太極拳與日常行走中。
  • 避免大腿蠻力：許多學員在坐下或起身時會慣性使用大腿撐起身體，導致肩膀僵硬。
  • 「溜滑梯」起身法：
  • 核心機制：膝蓋微往前移動，尾椎順勢像坐溜滑梯般下滑，帶動上半身從前傾轉為直立。
  • 重心帶動：利用身體的擺盪與重心移動來觸發動能，而非單純向上「抽起」身體。
  • 注意事項：身體不可一開始就往後倒，需維持重心在腳掌均勻分佈。

二、 站樁的三個層次

老師將站樁練習拆解為由外而內的進階過程：

  • 第一層次：架構（框架）
  • 雙腳平行與肩同寬，膝蓋微彎（不打直），收小腹、收尾閭。
  • 雙肩自然下垂，大拇指輕貼褲縫（確保腋下順暢），收下巴，舌抵上顎，頂頭懸。
  • 第二層次：肌肉（放鬆與均勻）
  • 尋找腳踝與小腿受力最均勻、最省力的位置。
  • 依序由下往上放鬆膝蓋、大腿、胸腔、背部、手臂、舌頭與臉部肌肉。
  • 第三層次：意念與狀態（外在連接）
  • 耳與眼：處於「等待」狀態。不主動抓取聲音或聚焦光線，讓資感自然進來，形成大螢幕般的開闊感。
  • 自然呼吸：不控制呼吸，單純「知道」氣息的進出與身體的聯動。

三、 步法與腳掌覺察

  • 腳掌均勻受力：無論是站立或折胯，腳掌必須均勻接觸地面。若受力不均（如重心全在後跟），會導致單腳站立或移動時失去平衡。
  • 抬膝練習：走路時練習「大腿提膝，小腿放鬆」，像有一條線牽著膝蓋提起。
  • 避免腳跟重著地：走路應學習先讓前腳掌輕輕著地，避免腳跟直接撞擊地面導致架構跑掉。

四、 肌肉調整與放鬆技巧

  • 自我按摩與拍打：
  • 使用「空掌」輕拍腿部，觀察肌肉是否會像果凍般「彈動」，以此觀測鬆的程度。
  • 撥筋：在肌肉放鬆後尋找深層的「筋」並進行撥動，其放鬆效益比局部按摩更大。
  • 呼吸減壓法：
  • 將注意力放在最緊繃的局部，觀察吸氣與吐氣時肌肉的微小膨脹與收縮變化。
  • 持續關注這種變化約一分鐘，肌肉會因為意念的注視而自然鬆開。
  • 肩頸拉伸：將重心移至一側，頭部斜向上延伸，利用身體自然重量產生的弧形來拉開肩頸，而非用手強拉。

五、 修練心法：Awareness（覺察）

  • 減少「在意」：不要太過在意別人的眼光或擔心自己做得好不好，這會導致身體緊繃。
  • 動態調整：站樁不是僵化不動。若感到某處僵硬到無法用意念調鬆，就應起身動一動，避免「硬熬」。
  • 觀察即是轉化：只要能「看到」或「感受到」僵硬，就是放鬆的開始。

一、課程主軸

  • 回顧坐姿與折胯的完整概念，導入走路基礎
  • 介紹站樁（小站樁）的三層次練習架構
  • 腿部調整的多種方法：捶打、按摩、垂拍、撥筋

二、坐姿（折胯）練習的再確認

【腳掌均勻接觸對打拳的意義】

後面學到單腳站立時，腳掌受力均勻 → 才能輕鬆地抬腳移動

如果重心都在腳跟，抬腳時壓力全在後頭 → 容易失去平衡

【練習核心】膝蓋以下幾乎不動，腳掌持續均勻接觸地面

上半身的折疊（折胯）和起身，全程腳掌感覺都不應該改變

「你腳掌那個接觸感，要從你開始往下的那一刻，一直到你站起來，它都是一樣的。這個才是練成的標準。」

【鬆的方式下去和起來】

鬆的下去：屁股放鬆往後，不是膝蓋蹲低

鬆的起來：溜滑梯感，不是大腿硬撐

→ 整個坐姿的練習，都在探索「用鬆來做動作」是什麼感覺

三、走路基礎——步驟拆解

【走路的兩個核心要求】

① 腳掌跟地面均勻接觸（不用腳跟著地）

② 膝蓋帶起，輕輕放下（不要踩踏）

【為什麼不用腳跟著地？】

腳跟著地 = 每一步都在撞腳跟

撞久了，腳跟的架構會跑掉（影響重心穩定）

先學習不撞，再慢慢學習整個腳掌的協調

【抬腳的正確方式】

大腿可以用力，小腿要放鬆

感覺像有一條線從膝蓋拉著腿上來，小腿是跟著的

放下時輕輕放，感受腳掌接觸地面的那一刻

「你能不能把腳輕輕地放下去，若有似無地這樣放著。它要承受全身的重量，但你在放的那一刻，是非常輕的。」

【重心的移動邏輯】

用「放鬆」觸發重心往前，不是用腳踢出去

前腳均勻著地 → 後腳輕鬆了 → 膝蓋帶起 → 輕輕放下

循環往復，每一步的差異極小 → 這就是走路版的動態平衡

四、站樁三層次——完整說明

【第一層：架構（外型正確）】

① 雙腳平行站立與肩同寬（腳趾朝前，非外八）

→ 如果習慣外八，可以腳跟往外翻再放下調整

② 膝蓋微彎（不打直就是微彎）

③ 收小腹、尾椎稍微往下（像準備坐下的感覺）

④ 雙肩自然下垂，大拇指輕貼褲縫

→ 這個位置讓腋下微開，不夾著

⑤ 收下巴、舌頂上顎、頂頭旋（脊椎延伸直立）

【第二層：肌肉均勻放鬆】

從腳踝往上，依序調整每個部位的肌肉：

腳踝跟小腿 → 膝蓋 → 大腿 → 臀部 → 腹部（稍微收住）

→ 背部可以放鬆；橫膈膜這一帶也可以鬆

繼續往上：胸腔 → 肩部 → 手臂 → 頸部 → 臉部肌肉 → 舌頭 → 後腦勺

→ 調完一遍後，再快速掃描一次，確認整體是均勻的

「如果說只打架構，那叫太極操。要開始加入第二層、第三層，才是太極拳。」

【第三層：感官等待（心的放鬆）】

耳朵：等待聲音進來，不是主動去抓

→ 先聽得到空調聲，再慢慢聽到外面的聲音，一層一層進來

眼睛：微微打開，讓光線自然進來

→ 不是聚焦看東西，是讓視野慢慢展開（像在電影院看大螢幕）

呼吸：自然呼吸，知道自己在呼吸就好，不要控制

→ 深淺快慢不拘，單純地知道「有氣在進去出來」

→ 進一步感受：每次呼吸，身體都有細微的變化

五、站樁的心態——允許調整，不硬撐

【允許晃動】

站樁時身體可能會微微晃動，這是正常的

不需要刻意穩住，讓它晃，觀察它為什麼晃（腿不穩？上半身緊？）

看懂後，再試著微調

【允許動一動再繼續】

如果覺得肩膀很僵、膝蓋很緊，就動一動再站

不是「站三分鐘不許動」的老式練法

「以前的練法是讓你站到把撐練到瓦解掉。那個非常辛苦！如果你練的是你把一個緊繃的狀態撐過去。這不是我們要的。我們要的是在站的過程中，持續調整到更鬆。」

【如何判斷要不要動？】

感覺到緊、不舒服 → 動一動，放鬆，再站

感覺身體在慢慢調整（微熱、輕微流動感）→ 繼續觀察

六、腿部調整方法——詳細說明

【按摩】用手指捏捏、抓抓大腿肌肉，找出哪裡緊

【捶打（握空掌）】

手握空空鬆鬆的（不要握緊），用這個掌輕輕捶打腿部

方向可以是前側、外側、後側

→ 捶打讓肌肉做被動的鬆緊交替

【垂拍法】

手臂抬起，瞬間加一點力往下，然後放掉

放掉之後讓它自然往下彈，不要拉它回來

→ 感受那個彈回來的力量：肌肉放鬆時，彈的感覺會更明顯

【撥筋法（進階）】

肌肉夠鬆後，可以感覺到皮下有一條筋

用大拇指關節輕輕撥那條筋，效益比按摩肌肉更大

撥一段，效果可以延伸到整條筋的長度

【床沿按摩（日常技巧）】

床墊邊緣剛好可以當按摩工具，躺著讓腿壓在床沿上滾動

全身都可以：腿的正面、側面、背部

→ 自己的身體重量是最自然的按摩力，痛感自己掌握

「做完大概一個鐘頭，精神就回來了。而且那個力量是你自己身體的力量，不會弄傷自己。」

七、問答重點節錄

  • Q：站樁時感覺很熱、有點癢，這是正常的嗎？

A：非常正常。熱和癢是氣血在流動的感覺。繼續站下去，等一下通常就會有一個清涼感進來。

  • Q：我站樁好像沒有什麼感覺，就是站著。

A：一開始很多人是這樣。你可以試著觀察：我的腳底是不是往前或往後？我的身體有沒有在晃？這都是一種觀察。有時候你觀察本身，就讓你開始有感覺了。

  • Q：練太極拳這幾堂課，感覺跟我以前學的太極拳完全不一樣。

A：以前的太極拳大多先教動作，再說放鬆。我們是先把放鬆練起來，再去做動作。這樣你之後學每一個動作，你都知道它在練什麼，而不是在複製外型。

八、課後練習建議

  • 每天做一次小站樁，從三分鐘開始，感覺可以延長就延長
  • 站樁後一定要走動，讓身體從靜態轉換回動態
  • 每天走路時練習：腳掌均勻接觸地面 + 膝蓋帶起輕輕放下
  • 腿部調整：練完之後做捶打或垂拍各 1-2 分鐘
  • 溜滑梯起身每天練 5-10 次，持續確認不是靠大腿撐起來
