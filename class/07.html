第七堂課

2026.04.03
本堂內容

一、本堂核心

這堂正式介紹站樁。站樁不是「站著不動撐著練」，而是透過三個層次——架構、肌肉放鬆、整體狀態——讓身體進入打拳前的最佳準備狀態。只做到架構就叫太極操；要到第二層、第三層，才是太極拳。坐姿讓你認識上半身放鬆的感覺，站樁就是把那個狀態帶著站起來。

二、站樁三個層次

第一層：架構 雙腳平行站立與肩同寬，腳跟微微往外（不外八），膝蓋微彎（不打直），收小腹收尾閭。雙肩自然下垂，大拇指側輕貼褲縫，讓腋下自然打開。收下巴（視線平視），舌頂上顎，頂頭懸（讓脊椎整條往上延伸）。架構確定之後不再動它。

第二層：肌肉放鬆 從腳踝往上逐層放鬆：腳踝 → 小腿 → 膝蓋 → 大腿 → 臀部 → 背部 → 腹部橫膈 → 兩臂 → 頸部 → 臉部 → 舌頭 → 太陽穴 → 後腦勺 → 頭皮。感覺不到就先知道有這件事，日後練到就調。調完後再快速掃描一次，讓各部位之間的力量均勻，沒有格格不入的地方。

第三層：整體狀態 耳朵：用「等待」的狀態讓聲音進來，不主動去抓。眼睛：讓光線自然進來，不聚焦，慢慢知道面前有什麼。最後知道自己在呼吸——不控制呼吸，只是知道有一口氣吸進去、一口氣吐出去，感受到每次呼吸身體都在微微變化。

三、站樁的做法提醒

僵了就動，不要硬撐。當你意念已經無法調整那個緊的地方，就要起身動一動、按摩，讓身體回到暢通狀態再繼續站。站樁的目的是進入放鬆狀態，不是為了撐住某個姿勢。

我們要呈現的是那個鬆的狀態。架構只是框架，然後加上肌肉的放鬆、整體的均勻，才是我們想要的。

四、走路練習正式化

腳掌均勻跟地面接觸，不要腳跟先撞地。膝蓋提起，小腿放鬆，輕輕放下。這個走法是把「腳掌均勻接觸＋鬆」帶進走路中的練習，一開始很慢很費力，但腿力和平衡感會慢慢累積起來。

五、回家練習

  • 站樁：三層次依序進入，第一層架構→第二層逐層放鬆→第三層耳眼等待；僵了就動，不硬撐。
  • 走路練習：腳掌均勻接地，小腿放鬆抬起放下。
  • 坐姿（複習）：坐完之後，把上半身放鬆的狀態帶進站樁，讓兩個練習互相驗證。
  • 腿部調整：按摩或空掌捶打疲勞的大腿、小腿肌肉，保持在鬆而非緊的狀態裡練習。

一、這堂課真正要練的是什麼

這堂正式介紹站樁，並把前六堂的所有練習整合在一起。

站樁不是「靜止不動地站著撐」，而是透過三個層次進入打拳前的最佳狀態：架構（骨架的位置）→ 肌肉放鬆（各部位逐一調鬆）→ 整體狀態（耳眼等待，呼吸流動）。阿郎說得很清楚：架構做到了只是太極操；要加上第二、第三層，才是太極拳。

同時這堂也正式把「走路練習」說清楚了，把折胯/坐姿累積的腳掌感受帶進日常行走。

二、坐姿練習的意義再說明

阿郎在課程開頭說了一個重點：坐姿的目的是讓身體體驗「上半身放鬆是什麼感覺」，然後把這個感覺帶進站樁和打拳。站樁的上半身狀態，就是坐姿最後上半身放鬆時的那個樣貌。所以如果站樁的時候上半身不對，可以回頭練坐姿，讓上半身重新學習那個放鬆的狀態，再複製過來。

打拳就是把身體端著走的放鬆狀態。

三、站樁第一層：架構

架構是整個站樁的框架，確定之後就不再動它，把注意力轉到第二層。

腳 雙腳平行站立與肩同寬。腳跟微微往外（讓腳掌更接近平行，不要外八）。可以用一個小技巧：腳尖抬起，腳跟往外翻再放下，就自然平行了。

膝蓋 微彎（不打直）。打直的膝蓋會讓力量撐在上面，容易僵化；微彎讓重力往下走。

腰腹 收小腹，讓尾椎稍微往下，有點像要準備坐下來的感覺。這讓髖關節微微翻正，腰椎也更接近直線。

肩膀與手臂 雙肩自然下垂，不前不後，轉動一下找到舒服的位置。大拇指側輕貼褲縫，讓腋下微微打開（不夾腋下），這樣手臂才能有自然的流動空間。

頸部與頭部 收下巴（視線從斜下方拉回平視），讓頸椎對齊脊椎。舌頭輕頂上顎。頂頭懸（讓脊椎整條往上延伸，頭頂像被一條線輕輕往上拉）。

四、站樁第二層：肌肉放鬆

架構確定之後，從腳踝開始往上逐層放鬆：

  1. 腳踝、小腿 → 找到均勻受力的位置，不要讓這裡積著太多壓力
  2. 膝蓋 → 放掉膝蓋的力量（架構還在，但肌肉不要緊抓）
  3. 大腿 → 放掉大腿的力量
  4. 臀部 → 稍微放鬆，但腹部還是保持微微收著
  5. 背部、橫膈膜附近 → 可以往下鬆
  6. 兩臂 → 讓整條手臂的肌肉都放掉
  7. 頸部、肩部 → 感受哪裡可以再調整
  8. 臉部肌肉 → 往內收（眉頭、下顎），不要用力
  9. 舌頭 → 輕頂上顎，不用力
  10. 太陽穴、後腦勺、頭皮 → 都是放掉的狀態

感受不到的地方不用勉強，先知道有這件事，日後練到感知更清楚再調。

調完之後快速再掃描一次：各部位之間的力量是不是「均勻流動」的，而不是「某段特別緊、某段特別鬆」形成格格不入的感覺。讓整體有一個暢通感。

五、站樁第三層：整體狀態（耳眼等待）

這是三層中最細膩的，也是從「練動作」到「練心態」的過渡。

耳朵：等待聲音進來 不主動去聽什麼，而是用「等待」的狀態，讓空調的聲音、外面的交通聲、阿郎的聲音，自然地進來。不抓、不找，讓它來。

眼睛：讓光線進來 微微打開，不聚焦，不掃視，讓光線自然進來。慢慢地，你會發現有亮有暗、有不同顏色、畫面慢慢展開——這都是光線自然形成的，不是眼睛主動去抓的。

呼吸：知道，但不控制 最後知道自己在呼吸。深淺快慢不拘，只是知道有一口氣吐出去、有一口氣吸進來。進一步，會發現每次呼吸身體都有微微的變化。看著這個變化，身體就不會那麼僵。

這個狀態就是：身體放在空間裡，心放在身體裡。

六、站樁的實際使用方式

站樁不是要撐，而是要在這個狀態裡微調、保持流動。

當你意念可以調整那個緊的地方，就繼續調整。當意念已經無法調整了，代表已經僵化，就要起身動一動，讓身體回到暢通再繼續站。強撐著僵化的身體繼續站，只是在練緊，不是在練鬆。

站樁的時間不用長，重要的是能不能進到那個放鬆流動的狀態裡。進去了，即使只站兩分鐘，也比站了半小時但一直僵著更有意義。

七、走路練習正式說明

這堂把走路練習說得更清楚了：

  • 腳掌均勻跟地面接觸（不要腳跟先撞地）
  • 膝蓋提起，小腿放鬆，輕輕放下
  • 用「放鬆的觸發」讓重心移動，而不是用肌肉力量推進

這個走法一開始很費力，因為你要有意識地維持每一步的均勻感。慢慢熟悉之後，腳掌的感受、平衡感、腿力都會跟著累積起來。

另外：起身的方式（前後腳擺盪起來）也在這堂繼續練習，讓膝蓋受力減少，靠重心移動自然起來。

八、腿部調整方法

練完蹲/走路之後，腿部一定會有壓力：

  • 按摩：用手指捏抓大腿、小腿肌肉，感受哪裡緊，幫助疏通
  • 空掌捶打：握空掌，輕輕捶打腿部，透過衝擊讓肌肉鬆動
  • 側邊壓著呼吸：把注意力放在緊的地方，觀察呼吸時那塊肌肉的變化，吐氣時它會微微鬆，持續一分鐘就會有效果
你的身體事實上有鬆的可能性。你光是關注它，然後發現它跟呼吸有關，仔細看著它就會鬆掉。

九、回家練習與下次銜接

  • 站樁：三層依序進入，僵了就動，不硬撐；站樁的上半身狀態從坐姿練習複製過來。
  • 走路練習：腳掌均勻接地，小腿放鬆，輕輕放下，用放鬆觸發重心移動。
  • 坐姿（複習）：完整練習一次，讓上半身記住放鬆的感覺，之後帶進站樁。
  • 腿部調整：練完後按摩或捶打，保持在鬆而非緊的狀態。
