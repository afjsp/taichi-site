2026.06.04
本堂內容

2026-06-04｜主題：心法先行，動作其次——翻掌心法與扭腰纏手至雙按

一、本堂核心

本堂以「心法先行，動作其次」為主軸。阿郎強調：少一堂課只是少幾個動作，只要心法在，回來就能接上。課堂分成翻掌心法的細部感受練習，與拳架複習（起勢至攬雀尾）加新動作組（扭腰纏手→順捛→盪手→當手→推轉掃→雙按）兩大區塊。下週休課，阿郎說：「課程不是一條直線，它是很多溪流跟大海，串流起來，只要能接回支流，就能跟上。」

「心法有了，我們再搭配動作。動作忘了還好，但是心法的部分要多多去熟悉它。」

二、翻掌心法暖身——把「多的」拿掉，讓身體自己動

這是本堂最核心的感覺練習，用握拳加力到放鬆的過程，體驗「鬆緊之間的覺察」。

核心流程：

  1. 握拳並逐層加力（手腕→下手臂→上手臂→肩膀）
  2. 逐層放鬆（肩膀→上手臂→下手臂→手腕→手指），力量拿掉，不需打開
  3. 此時手是僵化的，但比加力前稍鬆
  4. 掃描身體：頸部、肩膀、手——哪一邊相對鬆？
  5. 從鬆的那邊下手：輕輕移動，感受架構變化
  6. 給一點點意念，等身體自己動

關鍵心態：

  • 不要用意念控制整個肌肉去動，而是輕輕觸發，看身體是否回應
  • 找到鬆的地方就從那裡下手，不要從緊的地方下手
  • 若動起來有卡頓感，是意念過多——放手讓身體放鬆運作，卡頓就消失
「持續找鬆的地方下手，不要找緊的地方下手。」

三、拳架複習——起勢至攬雀尾

完整複習第一段前半，分三次練習：阿郎先打示範，學員跟打，最後一次自由練習。

動作序列： 起勢收肘→垂手→提腕→起手→伸指→屈臂→豎掌→伸手→沉手→合手→左云手捧→右云手捧→外轉云合→側上→左斜飛掤→轉腰→云合→側上→右斜飛攬雀尾

側上重點：

  • 腳要往旁邊拉開距離（步寬），不是放在身體旁邊，避免重心帶過去
  • 前後距離叫步長，左右距離叫步寬——此時要有步寬
  • 腳虛部放下：重心不移過去，腳才能出去又能回來
  • 扣腳角度：45度，前腳朝12點，後腳在3點鐘方向

攬雀尾重點：

  • 手勢保持開放，不要夾；拉近一點，只是單純碰著
  • 重心前七後三，身面朝著3點鐘方向

四、新動作——扭腰纏手至雙按

按步驟逐個練習，每個動作都以「腰帶、鬆手、最小力量」為核心。先做三次扭腰纏手，再做攬雀尾→扭腰纏手→順捛→盪手→當手→推轉掃→雙按全段。

  • 扭腰纏手：像翻煎蛋，從腰帶動手翻面，簡單轉過來就好；腰帶→手隨之翻→放鬆回來
  • 順捛：抓著往下拉的概念（捛、挒、濟、按四法之一）
  • 盪手：手肘在肩線前方，動作小，往外往上開；手肘絕不跑到後面去
  • 當手：手往外往上到耳朵高度，手肘幾乎不動
  • 推轉掃：推與轉同時，再掃開——擺脫被抓著的動作（rolling 的功能）
  • 雙按：手腕、手肘、肩膀均放鬆；從下往上比往前推更有聚力效果；重心前七後三

五、回家練習

翻掌心法是最重要的回家功課，每次練習三到五分鐘即可： 握拳→加力→放鬆→僵化後掃描→找鬆的切入點→最小意念觸發→等身體自己動。

不需要刻意練動作，心法熟了，任何動作都是在做新的練習。下次回來，只要接上心法，進度就能銜接。

2026-06-04｜主題：心法先行，動作其次——翻掌心法與扭腰纏手至雙按

一、本堂主軸

心法先行，動作其次。本堂以「翻掌心法」作為熱身感受練習，帶出觀察者心態；接續拳架複習（起勢至攬雀尾）並教授新動作組（扭腰纏手→順捛→盪手→當手→推轉掃→雙按）。下週休課，阿郎說：「少一堂課只是少幾個動作，心法一樣，接回來就是新的練習。課程是很多溪流跟大海，串流起來，只要能接回這個支流，就能跟上。」

整堂課的核心只有一件事：把多的拿掉，給一點點意念，讓身體自己動。

二、核心觀念——最小意念，讓身體自己動

太極的心法核心是：把「多的」拿掉，只留下最小意念，讓身體在鬆的狀態下自動運作。這不是放棄控制，而是換一種更精細的控制方式。

  • 意念不是「指揮肌肉」，而是「觸發身體去動」
  • 急的時候做不了任何改變；等到鬆的元素進來，變化才展開
  • 意念越多，卡頓越多；身體放鬆運作，動作才會流暢
  • 用「觀察者角度」：知道身體的狀態，讓身體帶你去，不是盯著死看
「光是把身體多的這部分拿掉的時候，你的心就會慢慢靜下來。要先把多的拿掉，後面才有空間讓它運轉。」

三、暖身感覺練習——握拳加力到鬆動身體

這個練習的目的：體驗「從僵化到鬆動」的完整路徑，讓身體學會「不是靠意志力撐著，而是靠鬆動改變架構」。

流程說明（約一分鐘內完成）：

  1. 握拳並逐層加力：手腕→下手臂→上手臂→肩膀，感受整個分段用力的狀態
  2. 逐層放鬆：肩膀→上手臂→下手臂→手腕→手指（力量拿掉，不需打開拳頭）
  3. 此時手是僵化的，但比加力前稍鬆——這是正常的，不要急著再開
  4. 掃描身體：頸部、肩膀、手——哪一邊相對鬆？從哪個地方可以動到這個區塊？
  5. 從鬆的那邊下手：輕輕左右移動，感受架構是否有一點不一樣
  6. 每動一次，觀察僵化部分是否稍有鬆動——找到切入點，讓鬆慢慢蔓延
  7. 給一點點意念：想讓它翻轉一些些，然後等身體回應

注意事項：

  • 僵化之後不要急著重新開啟，後面鬆的感覺就不見了
  • 不要找緊的地方下手，要找鬆的地方
  • 觀察者不是盯著看：是「知道他的狀態」，讓他帶你去到哪個地方
  • 這個工具平時也可以用：身體哪裡緊，走這個流程，不用給人家調，身體就會自己調整
「持續找鬆的地方下手，不要找緊的地方下手。」

四、拳架複習——起勢至攬雀尾

完整複習第一段前半，分三次練習：阿郎先示範，學員跟打，最後一次自由練習，跟不上就用眼睛跟、用想像跟。

動作序列： 起勢收肘→垂手→提腕→起手→伸指→屈臂→豎掌→伸手→沉手→合手→左云手捧→右云手捧→外轉云合→側上→左斜飛掤→轉腰→云合→側上→右斜飛攬雀尾

側上重點：

  • 腳要往旁邊拉開距離（步寬），不是放在身體旁邊
  • 前後距離叫步長，左右距離叫步寬——此時要有步寬
  • 腳虛部放下：重心不移過去，腳才能出去又能回來，不然就「摔出去」
  • 扣腳角度：45度，前腳朝12點，扣腳後腳跟到3點鐘方向，身面對準3點

攬雀尾重點：

  • 手勢保持開放，不要夾；距離不要太遠，拉近一點
  • 只是單純碰著這個位置
  • 重心前七後三，身面朝著3點鐘方向

五、新動作詳解（一）——扭腰纏手、順捛、盪手

扭腰纏手：

  • 核心：從腰帶動手翻面，像翻煎蛋一樣簡單——只是轉面而已
  • 翻的幅度少一點，腰稍微帶一下，手隨之翻，再放鬆回來
  • 正確做法：腰帶→手翻→放鬆回來；從另一角度看就是「扭腰」
  • 常見錯誤：夾著做（手肘往外張）；解法是把多的力量持續放掉
  • 到了這個位置之後，不同於云手的轉腰，因為已經扭著——要繼續折一下再放鬆回來

順捛：

  • 抓著往下拉的概念
  • 捛、挒、濟、按四個力法，順捛是下拉的運用
  • 重心在這裡從七三開始往後走到五五

盪手：

  • 手肘與身體幾乎沒有變化，保持在肩線前方
  • 動作小：往外開、轉起來——不要太豪邁（不是那麼大的動作）
  • 手肘一定在身體肩線的前面，不要跑到後面去——後面就沒有彈性了，被攻擊時就報廢
  • 更簡化版：直接在這邊往外收起來就好

六、新動作詳解（二）——當手、推轉掃、雙按

當手：

  • 手往外（手臂平的方向開出去）往上到耳朵高度
  • 手肘幾乎不動，降低後往外開起來就好
  • 常見錯誤：手肘跑到後面造成卡頓；手肘超過90度就太高了
  • 手指頭靠近耳朵這個高度即可，不用到頭頂

推轉掃：

  • 推與轉是同時進行的——推轉合一，再掃開
  • 功能：擺脫被抓著的狀態（rolling）；這個動作對被抓住的手有剝除效果
  • 掃手打開後，雙手放到腰間（不超過肩線，手肘留在身前）

雙按：

  • 手腕、手肘、肩膀均放鬆
  • 從下往上效果比往前推好：往前時對方可以順著動力飛出去；從下往上則可以「拔根」
  • 重心前七後三，到這裡輕輕一點就收了
「動作是排第二，心態放鬆是排第一。所以等一下的話，反正就還會拍影片，影片拍一拍，回去可以順著做。」

七、站姿調整與個別提醒

本堂阿郎逐一調整學員的攬雀尾站姿，重點包括：

步寬與步長：

  • 步寬（左右距離）要有，不要拉太窄——常見錯誤是步距長、步寬少
  • 步長（前後距離）可以稍短一點，不需要拉太長

肩膀與上半身：

  • 肩膀放鬆不是「往下壓」，是感覺能放掉的力量就把它放掉，吐一口氣有點沉澱下來
  • 若肩膀無法放，通常是頸部這邊的緊繃還沒鬆，頸部鬆了肩膀才能再往下沉
  • 身體不要往後倒，要往前折到位

後腳扣角：

  • 後腳扣進來的角度：45度，讓前後腳的夾角正確
  • 腳跟著地，前面腳掌轉進來就好，不用整個腳移動

站姿原則（打拳時通用）：

  • 打拳姿勢若蹲太低，膝蓋受力過大；建議膝蓋稍微高一點，避免在膝蓋還沒修正好前蹲太深
  • 上半身要往前折（不是往後倒），這樣重心才在前面七分

八、學員問答與回家練習

問答摘要：

  • Q：移動重心後再轉身，順序？ A：重心移過來，虛腳放出去，再彈過中轉正，平移到中間，再跨
  • Q：「沉」是什麼意思？ A：不只是往下，是放鬆加聚能，像彈簧壓縮——沉是有往下帶一點點，是聚能的意思
  • Q：吸氣吐氣的時機？ A：用力時吸氣，放鬆時吐氣；動作慢的話，一般呼吸就好，不用刻意配合
  • Q：承手跟合手有什麼不一樣？ A：承手是手掌上托的姿勢，合手是雙掌相對抱球的姿勢

回家練習： 翻掌心法是最核心的功課，每次三到五分鐘。流程：握拳→加力→放鬆→僵化後掃描→找鬆的切入點→最小意念觸發→等待身體回應。不需要刻意練動作，心法熟了，接什麼動作都是在做新的練習。下次回來，只要接上心法，進度就能銜接。
