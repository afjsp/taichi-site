第一堂課

2026.05.06
本堂內容

2026-05-06｜主題：脚底架構與重心轉移，從腳下開始讓身體學會流動

一、本堂核心

本堂從最基礎的「腳底站好」開始，讓每個人先感受自己與地面接觸的狀況。重心轉移不是靠推力，而是靠放鬆——放鬆哪一邊，重心就往哪邊流動。動作要慢，慢是為了讓你能看到自己肌肉的細小變化，身體才有機會自動調整。

二、脚底架構與均勻受力

脚底的骨架有三個支撐點：腳跟一個點，前腳掌那一排是一個面。肉的部分是感測器，要讓整個腳掌均勻貼地。可以試著重心稍微往前再往後，感受一下腳跟和前掌各自的壓力變化。

身體架構的部分在架構上面是呈現有一個支撐力的狀態，然後肉的話是均勻的貼地，肉是感測器。

站好之後，膝蓋要在腳趾正上方——不能往內夾，也不能往外偏，否則膝蓋打拳打越多傷越重。

三、用放鬆做重心轉移

要往左移，不是右腳用力推——而是把左側大腿前側、臀部下緣、髖關節放鬆一些些，重心自然就往左流動。另一側的髖關節放掉，就像放風箏的線，鬆一點重心就移一點。左右、前後都是這個原理。

前後移動時，用膝蓋稍微放鬆作為觸發點。這些左右、前後的移動本身就是一種感覺練習，然後再放鬆前後的胯，重心就會開始滑動。這個「滑動」的感覺，就是要慢慢練到手的狀態。

四、拳架複習：起勢

本堂練習太極拳起勢。口訣：

收肘 → 垂手 → 提腕 → 起手 → 伸直 → 屈背疏掌 → 伸手 → 成手 → 合手

重點是：手往上提的過程，找到肩膀開始「需要更多力量」的那個高度就停下來，不要硬撐。收回來時肩膀不聳，讓肩膀有向下的空間進來；伸手出去像海綿泡在水裡那樣輕輕延展，不是用肌肉推出去。

五、回家練習

  • 腳底練習：站好後慢慢把重心一點一點往前移，觀察哪條肌肉先動。越慢越清楚。
  • 重心轉移：左右、前後各練，用放鬆帶動，不用推。
  • 起勢：每次至少練三遍，肩膀不動為目標，找到每個動作肌肉開始「多」的位置。
  • 走路：帶著剛練完的流暢感走路，感受看看走路可以多放鬆。

太極拳的訓練核心不是學動作，而是學覺察。你慢慢動，看到肌肉在哪裡緊，然後放掉；你重心移動，感受那個滑動的感覺。這些都是在訓練你的「身體眼睛」，讓你越來越能看到自己。

2026-05-06｜主題：脚底架構與重心轉移，從腳下開始讓身體學會流動

一、這堂課真正要練的是什麼

這堂課不是教你記動作，而是讓你第一次認識「慢裡面能看到什麼」。課程從脚底開始——因為脚是我們與地面唯一的接觸點，把這個基礎搞清楚，重心轉移才有可能做對。太極拳的一切移動都建立在「放鬆帶動」而非「推力移動」，這個概念要從第一堂就開始身體記憶。整堂課的核心是：身體有自動校正的能力，而慢，是讓你看到它怎麼運作的工具。

二、脚底的架構：三點支撐與肉的感測器

脚的骨架有三個支撐點：腳跟是一個，前腳掌那一排骨頭是一個面。這是撐起全身重量的架構，是「點」跟「面」的概念。脚掌上的肉不是支撐者，而是感測器——它均勻貼地的時候，你才能感受到受力是否平衡。

骨架的部分是呈現有支撐力的狀態，然後肉的話是均勻的貼地，肉是感測器。

很多人走路腳外翻，外側壓力偏大，內側微微翹起，身體就不穩。感受的方法：重心非常緩慢地往前移，觀察哪條肌肉先動起來；再往後，感受腳跟接地的狀況。越慢、越輕，你看到的訊息越細。這個練習本身就已經是一種調整。

三、慢的道理：羽毛落在A4紙上

阿郎用一個比喻解釋為什麼要慢：一根羽毛落在A4紙上，你的手指能感覺到；但落在一塊方形磁磚上，你感覺不到——因為重量差太多，變化太小。同樣的，如果你的肌肉力量用太多，速度太快，你就看不到細小的肌肉變化。要能看到變化，就要把自己用的力量降下來，跟那個細小的變化「比例接近」，才看得到。

這就是慢的本質：慢不是目的，而是讓你能看到肌肉在每個角度的狀態，從而讓身體自行調整。身體看到了，它就會去修正。

四、重心轉移：用放鬆移動，不用力推

左右移動：要往左，不是右腳推——而是把左側大腿前側、臀部下緣放鬆一些，再把右髖關節（抓著你不讓你倒的那邊）也放鬆一點，重心就流過去了。就像放風箏：把線放鬆，風箏才往那個方向飄。要一點一點放，不要一次垮掉。

前後移動：前腿膝蓋放鬆是觸發點，啟動之後再搭配前後胯的放鬆，重心就會滑動。這個「滑動」感，就是太極拳移動的質地——不是跨步移動，而是滑過去的流動。

在這一邊放一點點，那身體不是從這邊這樣子移過來，然後再放這一側，不就是這樣子慢慢的移動嗎？

重心移完到另一腳，要回來：從肩膀這邊放鬆，搭配胯也松掉，身體就開始往回移動。整個過程的核心就是：把紧的地方找到，把它放掉。

五、膝蓋：只能在腳趾正上方

膝蓋不能往內夾，也不能往外偏，要在腳趾的正上方。內夾或外偏都會讓膝蓋吃力，打拳打越多傷越重。站重心的時候，先把膝蓋放鬆、讓它準備好要往那個方向移動，再把重心送過去，膝蓋就不會受傷。膝蓋在整個移動中是「承接者」，不是「推動者」。

六、手的練習：找卡點

雙手緩慢往上舉，觀察在哪個高度開始「需要比平常更多的力量才能繼續往上」——找到那個卡點，停在那裡。這個卡點就是肩膀肌肉不均勻的位置，也是你需要調整的地方。

到這邊的時候突然覺得我肩膀需要有再多一點的力量他才起得去，那你就停留在這裡。

從那個卡點開始，試著找哪一邊可以放鬆——膝蓋稍微松一點、背部松一點——就可以繼續往上。這說明「舉手」這件事不只是手的動作，而是整個身體狀態的呈現。

七、太極拳三要素：鬆緊、呼吸、動態平衡

阿郎點出太極拳的核心有三個元素：

  • 鬆緊：找到緊的地方，調鬆，讓移動發生。
  • 呼吸：光是呼吸就可以讓肌肉放鬆。緊的地方隨著呼吸有張縮，你只要停留在「不那麼緊」的位置，它就會慢慢鬆下來。
  • 動態平衡：一直在做重新的平衡，就是動態平衡。

這三個串起來，就是太極拳每個動作的本質。

八、觀無極：打拳之前的身體掃描

站樁後進入觀無極，有三層：

  1. 骨架：雙腳平行與肩同寬，膝蓋微彎，收小腹，肩膀自然下垂，收下巴，舌頂上顎，頂頭懸。
  2. 肌肉：從腳踝往上，一層一層掃描肌肉，找到壓力最小、最均勻的重心位置。
  3. 意念：眼睛不刻意聚焦，像等待光線進來；耳朵等待聲音，不主動去抓；呼吸自然順暢。這是「等待的狀態」。

把自己調到這個均勻的狀態後，打拳的每個動作都要能維持這個狀態——這就是太極拳難的地方，也是它練不完的原因。

九、起勢動作拆解

收肘 → 垂手 → 提腕 → 起手 → 伸直 → 屈背疏掌 → 伸手 → 成手 → 合手

  • 收肘：手肘往側邊收，不往後收（往後收容易被夾住）。手掌滑過一個球面。
  • 垂手：手掌下垂，大拇指輕貼褲縫旁邊。
  • 提腕：手腕提起，像拿著一根藤條把腕抬起來。
  • 起手：手往上，感覺卡點在哪。
  • 伸直：手指頭往前延伸，不是打直，是延伸，手掌朝前平放。
  • 屈背疏掌：手肘往下折，手掌像翻起來——但不要用力翻，讓它「泡水海綿」般展開。
  • 伸手：手掌往前平推，背部有延展感，不是肌肉推出去。
  • 成手：手肘順著放下。
  • 合手：像抱一個大球在小腹前，腰的高度，不要太往前。

肩膀全程不聳，手的動作盡可能由身體帶，不是肩膀帶。

十、學員問答

Q：膝蓋微彎的「微」是多少？ A：就是從打直稍微放掉，非常輕微的彎——不是往內夾，就是輕輕一放而已，這就夠了。

Q：重心在某一腳之後怎麼回來？ A：從肩膀這邊先動一下，搭配胯也松掉，身體就開始往回移動了。

Q：為什麼要這麼慢？ A：慢不是目的，覺察才是。快的時候你看不到細小的肌肉變化；慢了，你才能看到，身體才有機會自動調整。你在練的是覺察，動作只是工具。

十一、回家練習與下次銜接

  • 腳底均勻受力練習，每天走路都可以做。
  • 重心轉移（左右、前後），用放鬆帶動，不用推。
  • 起勢重複練習，肩膀不聳、伸手不用力為目標。
  • 坐姿練習（折胯）：讓上半身能在坐姿中保持放鬆，這是下次站樁的前置條件。

下次課程會把坐姿作為基本功帶大家一起做，繼續深化本堂的重心轉移概念。
