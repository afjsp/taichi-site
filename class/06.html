第六堂課

2026.03.27
本堂內容

一、本堂核心

這堂把坐姿完整說明了。坐姿不是練習端正坐著，而是讓身體學會「上半身放鬆、壓力從腳底洩出去」的狀態——這正是打拳時上半身需要呈現的基礎狀態。從折胯觸椅開始，透過一層一層放鬆腿部，讓上半身自然直立起來，最後四個支撐點（坐骨×2 + 腳掌×2）讓壓力完全流到底部。

動作只是在協助我們來調到鬆的狀態，所以不要誤解，動作只是來協助而已，我們要得到的是那個鬆的狀態。

二、坐姿的完整步驟

折胯到觸椅（不要坐下） 椅子距離一個腳掌，用折胯的方式慢慢往後，屁股輕觸椅面後停住。不要把重量全部放在椅子上——那樣壓力就散掉了，後面的練習就失去意義。

手放下，一層一層往上放鬆 手放下來，全部壓力在腿上。接著從腳趾往上逐層觀想放鬆：腳趾 → 腳踝 → 小腿 → 膝蓋 → 大腿 → 胯。每放一層，上半身就微微直立一點。整個過程是「腿的壓力釋放 → 上半身自然起來」，不是靠上半身的力量挺直。

腳掌往前，三點一直線 最後把腳掌往前放：讓髖關節、膝蓋、腳踝呈一直線，壓力才能順暢往下流。這樣上半身的壓力就完全往下倒，坐骨+腳掌的四個支撐點承受全部重量，肩膀和手就完全沒有壓力了。

三、起身的方式

一隻腳往前、一隻腳往後，用擺盪的方式起來：重心往前、膝蓋往前繼續頂，前腳稍微撐住高度，後腳只是過客。擺盪讓膝蓋不用持續受力，越擺越高，最後收腳站起。

四、重要觀念：練鬆不練緊

你要練的是鬆的感覺，不要說我要為了練了這個姿勢的熟練，然後但是你得到一個緊的結果，那個不是我們要的。

練習要練「鬆的感覺」，不是練熟一個動作的外型。每次練習時間不用長（一兩分鐘），但頻率多。帶著疲勞繼續練只會越練越緊，要在身體還放鬆的狀態下練。

五、回家練習

  • 坐姿：折胯觸椅停住 → 手放下 → 從腳趾往上放鬆 → 腳掌往前三點一直線 → 感受四個支撐點，上半身自然放鬆。
  • 起身練習：一前一後腳，用擺盪方式起來，不用力頂。
  • 折胯蹲練（複習）：腳掌均勻接觸地面，臀部往後＋上半身往前，切細步驟慢慢做。
  • 頻率多、時間短：每次一兩分鐘，感覺到鬆的狀態就可以停，不要撐到疲勞。

一、這堂課真正要練的是什麼

這堂完整教授坐姿，同時把前幾堂的所有練習（站立重心、折胯蹲練、呼吸、鬆緊）整合在一起。

坐姿的目的不是學習一個「正確的坐法」，而是讓身體記住「上半身放鬆、壓力從腳底洩出去」的這個狀態——這正是打太極拳時上半身需要呈現的基礎狀態。

打拳就是把身體端著走的放鬆狀態。

阿郎也說了一個重要的反覆原則：練習是拿來練「鬆的感覺」，不是拿來練熟一個動作的外型。帶著疲勞繼續練，只會累積緊繃；每次練習時間短、頻率多，保持在可以感受到鬆的狀態裡才是真正的練習。

二、折胯蹲練的再次精修

這堂繼續修正了幾個常見問題：

臀部往後是「被拉著」，不是主動蹲 臀部（髖關節）往後的感覺，比較像是有人從後面把這裡往後拉，而不是自己用力蹲下去。這樣帶動的是整個髖關節，上半身才能自然地對等往前，而不是各自分開動作。

腳掌是重心的測驗儀 在折胯的過程中，腳掌要維持均勻接觸地面——這是判斷重心有沒有跑偏的依據。腳跟起來代表重心往後太多（上半身給的不夠）；前腳掌壓力大代表重心往前太多（上半身給太多）。

慢，要切細步驟 動作要切成十五等份，幾乎感覺不到自己在動，這樣才能在每個過渡點感受到力量的變化，做出調整。

三、坐姿完整流程

步驟一：折胯到觸椅（不要坐下）

椅子放在背後約一個腳掌距離。用折胯的方式慢慢往後，屁股輕觸椅面後停住。

為什麼不能直接坐下去？ 一坐下去，壓力就全部散掉了，後面的「一層一層放鬆腿部讓上半身直立」就失去了練習意義。輕觸後停住，這時候全部的壓力都積在腿上——這是「充電」的狀態，接下來的練習才有東西可以釋放。

步驟二：手放下，從腳趾往上一層一層放鬆

把手放下來，讓全部壓力留在腿上。接著從腳趾開始，往上一層一層觀想放鬆：

  1. 腳趾、腳掌放鬆 → 臀部更踏實地貼住椅面
  2. 腳踝放鬆 → 上半身開始有些微變化
  3. 小腿放鬆 → 肩膀開始鬆開
  4. 膝蓋放鬆 → 上半身慢慢往直立的方向移動
  5. 大腿放鬆 → 繼續直立
  6. 胯（髖關節）放鬆 → 肩膀整個鬆下來

每放一層，腿部的壓力釋放，上半身就自然地微微直立一點。這個過程是「腿的壓力帶動上半身自然起來」，不是靠上半身主動挺直。這跟小腿拉筋時「從小腿往上一層一層放鬆，拉筋越來越深」是完全相同的邏輯。

步驟三：腳掌往前，三點一直線

這時候壓力還沒完全往下流。最後一步是把腳往前放：

讓髖關節 → 膝蓋 → 腳踝 → 腳掌呈一條直線（就像水管不要折到）。這條線對齊了，上半身的壓力就可以完全往下流，坐骨的感覺會非常明顯，就像放在上面的那種紮實感。

最終狀態：坐骨（兩點）+ 腳掌（兩面）= 四個支撐點，上半身完全沒有壓力，肩膀手臂自然放鬆下垂。這就是打拳時上半身的狀態。

脊椎的架構維持（脊椎是直的），但肌肉放鬆——架構在，肌肉不緊，這是關鍵的區別。

四、起身方式：擺盪起來，不用力頂

起身的方式用擺盪，讓膝蓋不需要持續受力。

一隻腳往前、一隻腳往後，重心往前，上半身順著往前傾帶動重心移動。前腳稍微頂住高度，後腳只是過客。然後膝蓋放鬆，重心稍微往後，再往前，這樣前後擺盪，身體越來越高，最後收腳站起。起身那一刻，重心主要在前腳。

這個方式讓膝蓋的受力分散，不是一次用力撐起。

五、練習觀念：練鬆不練緊

阿郎特別強調了一個反直覺的重點：

帶著疲勞繼續練，只會讓肌肉越練越緊，因為身體在緊繃的狀態下，力量會一直累積。我們要練的是鬆的感覺，不是練熟一個動作。

你要練的是鬆的感覺，不要說我要為了練了這個姿勢的熟練，然後你得到一個緊的結果，那個不是我們要的。

正確的練習頻率：每次一兩分鐘，頻率多。感覺到鬆的狀態就可以停，不要撐到疲勞。這樣累積的是鬆的記憶，不是緊的習慣。

六、坐姿的意義：打拳的基礎

阿郎說了一個很清楚的原因為什麼要學坐姿：

當上半身在這個放鬆的狀態時，打拳的時候才能「用腰來帶動手」——你只要轉個腰，意念一動，手就可以配合起來，不需要太多的力量。如果上半身卡著，就只能靠蠻力做動作。

所以坐姿練習的目的是：讓身體記住「上半身放鬆是什麼感覺」。記住了，之後不用每次都做完整的坐姿練習，只要在打拳前把這個狀態喚回來，上半身就能保持放鬆的狀態在動作裡運作。

收下巴（讓頸椎對齊脊椎）也是這個整體狀態的一部分——下巴微收，視線平視，頸椎直了，從坐骨到頸椎的整條路徑才真正暢通。

七、學員問答重點

有學員問腳掌外圍（足弓不夠）的問題。阿郎說：踩球（彈珠）可以幫助足弓撐起來，改善腳掌架構；蹲姿練習加上手肘撐開膝蓋，防止 X 腳，也是同一個方向的調整。

有學員說坐姿坐不久。阿郎說：正常，因為不習慣。不是姿勢不對，而是身體還沒有習慣這個狀態，習慣了以後自然就可以坐很久，而且很舒服。

八、回家練習與下次銜接

  • 坐姿：折胯觸椅停住 → 手放下 → 從腳趾往上逐層放鬆 → 腳掌往前三點一直線 → 感受四個支撐點，上半身放鬆。
  • 起身：一前一後腳，前後擺盪起來，不要一次頂起。
  • 折胯蹲練：繼續練，切細步驟，感受腳掌均勻接觸地面。
  • 頻率多時間短：每次一兩分鐘，保持在鬆的狀態裡練，不要帶著疲勞繼續撐。
