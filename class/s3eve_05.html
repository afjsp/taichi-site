第五堂課

2026.06.17
本堂內容

2026-06-17｜主題：鬆是心法不是放掉力量——握拳鬆放找切入點，複習第一段拳架，新教扭腰纏手到進身雙按

一、本堂核心

這堂課真正在練的只有一個字：鬆。阿郎一直強調，鬆不是把力量整個放掉，而是在架構維持得住的前提下，把多餘的力量一點一點拿掉，讓身體能自己流動、自己帶著動作走。

整堂從握拳鬆放、站樁的觀念入手，再回到拳架，把第一段複習到攬雀尾，接著新教攬雀尾之後的「扭腰纏手→順捛→盪手→轉正補擠→平坐回身分提屈臂→進身雙按」。無論練功或打拳，回到的都是同一句話。

最後的目標就是要把多餘的力量拿掉，講到哪裡，就都用這個態度、這個心法加進來。

二、鬆的入口：握拳鬆放練習

握拳，依序把手腕、下手臂、上手臂、肩膀的力量加上去，再反過來一層層放掉。力量一放掉，先感受到的是「僵化」——僵化是固態、不太能動的狀態。

重點在於：把力量放掉，不等於鬆。鬆是有切入點的，不是整隻手平均放鬆，而是先找到手臂裡相對比較靈活、比較自在的那一塊，從那裡開始用「鬆的動作」慢慢調整其他緊的地方。找不到時，甚至改變一下呼吸，狀態也會跟著被發現。

有點像是通水管的過程，慢慢把阻塞的地方消掉。每個人狀況不一樣，不用比較，能抓到一點就持續在上面熟悉。

三、站樁與「層層」的概念

站樁不是用腿去「撐」，最容易犯的錯就是把力量撐在那裡。要把腿部的肌肉力量縮到剛剛好，架構到位、肌肉放鬆，這樣站才養得出元氣。身體不要太挺，太挺橫膈膜沒有空間、呼吸會又快又緊；微微鬆沉下去，上半身就多出空間，呼吸自然順。

雙手端著、慢慢抽掉力量往下掉的過程，會感覺到一層一層的鬆，這個「層層」要用到全身。

剛剛好你再多一點點，就是肌肉整個把它舉起來；再少一點點，它就往下掉。在這中間把全身都調到這個狀態。

四、拳架複習與新動作

第一段複習（起勢到攬雀尾）：起勢序列為收肘、垂手、提腕、起手、伸指、屈臂、豎掌、伸手、沉手、合手；接左雲手掤、右雲手掤、外轉雲合、側上、左斜飛掤、轉腰雲合、側上、右斜飛、攬雀尾。提醒重點：

  • 側上要「留有步寬」，腳是踏到旁邊，不是踏在正下方；這是本段最常忘的地方。
  • 外轉用腳跟轉，方向大約轉到三點鐘（90度）；反架（向南）則左右對調，從左胯啟動。
  • 後腳扣回來用身體帶，維持前七後三；手的腋下要留空間，不要伸太遠。
  • 左／右雲手「掤」手約垂直、平行地面，只有一點點，不是放開來的大動作。

新教：扭腰纏手到進身雙按。盪手是本堂重點，先學固定版：手肘幾乎不動，只有下半部（下手臂、手腕）往外往上到耳朵，手肘不要靠到身上、也不要太遠離身體（用比「6」的距離抓在約45度斜出去的位置）。熟了再加上弧度，並進一步用身體沉胯來帶，而不是用肌肉舉。

往外還沒打直的時候再往上，更容易直接上；手肘一定不能打直，打直之前就要已經上了。

五、回家練習

  • 握拳鬆放：每次約5分鐘加3分鐘，重點不是動作，而是看清楚從緊到鬆的路徑——哪一邊先鬆、哪一邊還卡。
  • 站樁：手合著慢慢站，全身掃多餘的力量，腿力縮到剛剛好，養元氣。
  • 盪手：固定手肘、抓45度位置，先求動作小而慢不卡，再練用身體帶。
  • 心法：任何時候、任何動作，定著不動時也先掃一遍多餘的力量，再看身體要怎麼動。要往哪邊移就鬆哪隻腳。

2026-06-17｜主題：鬆是心法不是放掉力量——握拳鬆放找切入點，複習第一段拳架，新教扭腰纏手到進身雙按

一、這堂課真正要練的是什麼

這堂課的重點，不是「多學會幾個招式」，而是把「鬆」這件事練清楚。阿郎反覆說明：鬆不是把力量整個放掉，而是在身體架構維持得住的前提下，把多餘的力量一點一點拿掉，讓身體能自己流動、甚至自己帶著動作走。

整堂的安排，是先用握拳鬆放、站樁把「鬆」的身體感受建立起來，再回到拳架，把第一段複習到攬雀尾，接著新教攬雀尾之後的一整段：扭腰纏手 → 順捛 → 盪手 → 轉正補擠 → 平坐回身、分提屈臂 → 進身雙按。練功和打拳看起來是兩件事，但回到的都是同一個心法。

最後的目標就是要把多餘的力量拿掉，講到哪裡，就都用這個態度、這個心法加進來。

二、鬆不是放掉力量：從緊知道鬆

課程一開始，學員談到站樁與拉筋的經驗。阿郎點出一個關鍵：硬拉、用意志力放手，往往放不掉；真正有效的是「等待身體自己放下去」，這種放鬆來得慢，但效果比較持續。要鬆，得先從「緊」開始認識——身體要從緊的狀態裡，慢慢適應、慢慢往下一個階段走，需要時間，不能急。

這也帶出本堂最重要的分辨：「把力量放掉」和「鬆」不是同一件事。力量放掉之後，第一個感受到的常常是僵化；僵化是固態、不太能動的狀態。鬆則是靈活、有彈性的，而且它是有切入點的，不會整塊平均地鬆開。

三、握拳鬆放練習：找到鬆的切入點

具體練法：握拳，把手腕、下手臂、上手臂、肩膀的力量「一層一層」加上去，去感受力量加上去時整個狀態的改變；接著反過來，從肩膀、上手臂、下手臂、手腕到手指，一層層放掉。

放掉之後不要急著做大動作，而是先觀察：整隻手臂裡，哪一塊是相對比較靈活、比較自在的？從那一塊當切入點，用「鬆的動作」（靈活地微微轉動，而不是僵著硬轉）慢慢去鬆開其他比較緊、比較卡的地方。阿郎也提醒，這個過程裡甚至改變一下呼吸，原本沒注意到的狀態也會浮現出來。

練到後面，重點是「跟著身體走」：當你把多餘的力量拿掉、不主動施力時，去看身體會不會自己有一個驅動力把動作帶起來。每個人通暢的程度不同，不必互相比較。

有點像是通水管的過程，慢慢把阻塞的地方消掉。還沒那麼通順沒關係，能抓到一點就持續在上面熟悉，就能慢慢跟上。

四、站樁與「層層」：架構到位、肌肉放鬆

站樁這段，阿郎先破除一個常見誤解：站樁最容易犯的錯，是用腿去「撐」——撐就僵化了。正確的是把腿部肌肉力量縮到剛剛好，架構（外形、關節位置）要到位，肌肉卻要放鬆。這樣站才不是耗力氣，而是養元氣，但要站到位需要花時間慢慢調。

身體的「挺」也要調整。太挺的時候，橫膈膜沒有空間，呼吸會又快又緊；微微鬆沉下去，上半身反而多出空間，呼吸順、也更容易做細微的覺察。

接著是「層層」的概念：雙手端著、慢慢把力量抽掉往下掉，會感覺到一層一層的鬆，這個層層要遍佈全身。阿郎用一個很具體的描述抓這個分寸——

剛剛好你再多一點點，就是肌肉整個把它舉起來；再少一點點，它就往下掉。架構都一樣到位，肌肉放鬆，把整個身體都調到這個狀態。

此外，若覺得「肩膀很重」，多半是重心沒找到平衡點，或處在緊張時的「備戰狀態」（一有事肩膀就聳起來）。這種狀態要花比較久去觀察，先把裡面可以調的多餘力量調掉。

五、第一段拳架複習（起勢到攬雀尾）

本段把第一段打到攬雀尾，完整序列為：起勢（收肘、垂手、提腕、起手、伸指、屈臂、豎掌、伸手、沉手、合手）→ 左雲手掤、右雲手掤 → 外轉、雲合 → 側上 → 左斜飛掤 → 轉腰雲合 → 側上 → 右斜飛 → 攬雀尾 → 向前收勢。現場修正重點：

  • 側上的步寬：這是最常忘的地方。重心站上來之後，腳不是踏在正下方，而是要踏到旁邊，留有步寬；側上時腳提起來往旁邊放，重點都在「步寬」。
  • 轉腰雲合：45度角扣進來，一樣要留有步寬。
  • 後腳與重心：搭上來時後腳要扣回來，用身體帶，維持前七後三；手的腋下要留空間，不要伸太遠，拉回來一點。
  • 外轉方向：用腳跟轉，方向大約到三點鐘（90度，以12、3、6、9點定位），原地外轉、後腳不必另外轉45度。
  • 雲手「掤」：要分清楚「雲手」和「掤手」——掤手約垂直、平行地面，只有一點點，不是放開來的大動作。從左胯啟動，腰鬆一點點，往內收（左雲收掤、右雲收掤）。若打反架，就是左右對調，從左胯啟動、整個動作朝左邊運行。
  • 攬雀尾：不要繃開了再拉進來，手不是直的，是有收的。

六、新教動作：扭腰纏手 → 順捛 → 盪手

攬雀尾之後接新的一段。盪手（本堂花最多時間）是關鍵，阿郎拆成幾個層次來教：

先學固定／簡易版：手放到胸前做一個掤手，單純往前放、還不到打直，就把它舉起來到耳朵。手肘幾乎不動（在伸直的地方），真正動的是下半部——下手臂和手腕，上半部只轉一點點。常見錯誤是手肘亂跑：手肘不要靠到自己身上（會塞住、夾住），也不要離身體太遠（會整個夾起來），要抓在約45度斜出去、離身體近一點的位置。阿郎用比「6」的手勢來抓手肘到身體的距離。

再學有弧度的正常版：平開之後往上到耳朵，過程一直轉，走出一個斜的小弧，「像是一個肘」這樣滑上來。關鍵是手肘一定不能打直。

往外還沒打直的時候再往上，更容易直接上；手肘一定不能打直，打直之前就要已經上了。

最後加上身體帶：熟了之後，不要再用肩、用肌肉把手舉起來，而是用身體律動——沉一下胯，讓手自己被帶起來（和「蹲下去、守起來」是同一個概念）。要做到這層，得先把撐住的、多餘的力量卸掉，動作才有流動性。練的方法是「慢」：慢到幾乎要停下來，就看得到哪裡還有肌肉在夾，把多餘的拿掉，身體就安靜下來，再看它自己怎麼動。雙手可以互相對照：哪隻手順，就跟那隻手學。

七、轉正補擠到進身雙按、重心變化

盪手之後接：轉正 → 向前補擠 → 平坐回身、分提屈臂 → 進身雙按 → 坐掌。

  • 轉正、向前補擠：轉正後，手跟重心同時往前補擠。補擠裡帶一個「Rolling」（轉動）的概念，是發勁會用到的東西，是「一推一轉」。阿郎用拆解來說明應用：人家抓著你的時候，靠轉動鬆開、把對方的手剝掉，所以整個是「推、轉、掃」。掃完，身面對準前腳。
  • 平坐回身、分提屈臂：重心往後，兩邊分提、屈臂，手背彎曲往下放到腰間。提醒：不是手肘往「後」，而是手肘往「側」，再從下方起來。
  • 進身雙按、坐掌：從腰間往前、往上做雙按，起到肩膀高度，再放鬆肩膀、手肘、手腕，坐掌收尾。坐掌時不要把手腕翹起來（不要有那個「橫」），這邊幾乎是平順的，放倒來看也只有一點點。

重心變化（這段的骨架）：起手前七後三 → 扭腰纏手仍是七三，只是腰輕轉一下彈回來 → 順捛時重心開始往後到五五 → 做掤手、準備盪手，盪手往外轉時到六四或七三、重心已往另一邊 → 轉正、身面轉回來到五五，手搭上去約前六後四 → 往前七三 → 重心往後五五，手放下到腰間 → 再往前回到五五、七三做雙按。要訣是：往哪個方向移動，就先鬆哪一隻腳。

八、學員問答

  • 「重心鬆不動怎麼辦？」：阿郎回到本堂主軸——鬆不動，表示哪一邊僵掉了，那一邊就是要練的地方。先去找哪一邊相對鬆、相對有彈性（例如前腳膝蓋常常是相對能活動的），讓它鬆一點，架構只要改變一點點，原本僵掉的地方就跟著鬆動，再繼續走。
  • 「身體僵掉了，怎麼解？」：通則和上題一樣——看哪邊是鬆的、最有彈性的，從那裡切入。這個方法任何僵掉的情況都適用。
  • 「攬雀尾在45度，盪手要注意什麼？」：因為攬雀尾這隻手停在45度、手肘幾乎不動，所以接盪手時，手的位置其實已經差不多準備好了，是身體最省力的位置。手肘和身體的關係維持不動（身體可以轉，但手肘沒有大幅往外的動作），盪手就會輕鬆很多。
  • 反架（向南、相反方向）：不確定哪邊做得好時，就左右互相觀察——先做順的那隻手，看它為什麼順，再回頭看另一隻手哪裡卡、哪裡用力，問題就出來了。

九、回家練習與下次銜接

  • 握拳鬆放：約5分鐘加3分鐘。重點不是動作本身，而是看清楚「從緊到鬆」的路徑：哪一邊先開始鬆、哪一邊還卡，慢慢把多餘的散掉、回到沒用力的原始狀態。
  • 站樁：手合著慢慢站，全身掃多餘的力量，腿力縮到剛剛好、不要撐，養元氣。
  • 盪手：先求動作小、慢、不卡——固定手肘、抓45度斜出去的位置，手肘不要太近也不要太遠；熟了再加弧度，再練用身體沉胯來帶。
  • 心法總綱：任何時候、任何動作，甚至定著不動時，都先掃一遍多餘的力量、把它拿掉，再看身體有沒有訊息告訴你怎麼動。一次抓一個重點就好，不用一次全都顧到。
我常常會說，我們就是回到原點——你只要記得一個動作，就是把多餘的力量拿掉，然後講到哪裡都用這個心法加進來，這樣就好。
