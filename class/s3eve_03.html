三

第三堂課

2026.05.20
本堂內容

這份簡易要點根據課堂記錄整理（本堂無音檔），讀的時候不用先背動作，先抓住本堂的主線。

一、起勢：在慢之中覺察

起勢的過程要慢。在慢的過程中，先讓自己能察覺到肩膀是鬆的還是緊的。

如果是在緊的範圍內，看可以調整到什麼程度——能力範圍內的就去做；不是自己能調整的範圍，就試著在這個區塊找到「氣」的元素，再進行調整。

核心環節：起勢中手腕「提起來」與「放下來」的環節是重心所在。不是要用一次呼吸跑完整個流程，而是感受如何用最小的力量運作，在最小力量的狀態下做任何動作。

二、左雲手掤、右雲手掤：手要「合」、腰只微移

轉腰的真相：事實上只有胯跟膝蓋部分會前後略微移動，並不會轉動到腳，也不要讓它變成螺旋狀的旋轉。

輕微地移動膝蓋，做前後的移動而已，不刻意「轉」。

三、轉腰雲合：腳跟著地、腳掌轉動

右腳的轉法：腳跟保持著地，腳掌提起，隨著腰胯轉動往外轉；轉到 90 度時，把腳掌放下來，重心往前。

收回的細節：收回來的時候，胯要放鬆，右腳腳掌提起並轉回來。單純練習重心站上去，再把重心收回到左腳。

四、左右轉的基本功

右轉：重心移動到右腳 → 左腳腳掌提起 → 左胯放鬆並往左邊轉動 → 左腳腳掌落下 → 重心站上去 → 左腳腳掌再提起、左胯放鬆轉回來。

左轉同理，兩邊互為鏡像。先把左右轉的基本功練順，後續才能延伸到「提起腳往前移動」。

五、提起腳往前移動：回到足弓三原則

在移動腳往前時，留意足弓三原則：

  • 腳底要能與地面均勻接觸。
  • 腳跟與前腳足弓前方的骨頭區塊，是均勻支撐重量的地方。
  • 腳步踏穩，上半身開始放鬆。

六、過中轉正：平移後再旋

移動到中間時是平移的狀態，到中間之後，用向左旋或向右旋的方式，讓重心持續往前，但身面會轉到對準前腳的位置。這是「左斜飛掤」（以及後續類似步法）的腳步走法。

七、左斜飛掤與合手

手的部分：左手會一起帶起來，變成「掤手」。在轉腰雲合、過中轉正的過程中，會「合手」。

合手的形狀：上下相對、掌心相對——有可能是右手在上，也可能是左手在上。相對之後，大部分時間手都會守在身體的中間。
