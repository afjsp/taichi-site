2026-05-14｜主題：拉筋入門——從緊中看見鬆的可能

一、本堂核心

本堂主要引入「拉筋」系列的動作，作為覺察練習的另一個入口。拉筋的目的不是把筋拉得多大，而是先進入一個「緊」的狀態，然後透過慢和呼吸，觀察那個緊怎麼一點一點鬆開。起勢也再次複習，增加了「伸手時搭配膝蓋放鬆」的細節。

二、拉筋的邏輯：緊是入口

拉筋是讓你先到達一個緊的邊界，在那個邊界停下來，然後用呼吸讓它慢慢鬆。緊的時候放掉一點點，感覺就很明顯——這就是進入覺察的一個好用方式。

手腕轉向後的動作：手不打直，微彎放著，手指左右延伸，然後手掌緩慢往後翻。翻到肩胛骨開始有感覺的時候，加上收下巴、後腦勺稍微往後，延展會更明顯。在那裡停著呼吸，吐氣時鬆一些，它就慢慢往下走。

緊的狀態是入口，從那裡慢慢往鬆的方向走。

三、慢的練習：幾乎感覺不到它在動

這個「放下的練習」是本堂最核心的感覺練習。把手放到一定高度，然後「只放一點點支撐的力量，讓它幾乎要往下掉但又沒有掉」。在這個狀態裡，肌肉的變化非常細微，但你是能感覺到的。這就是覺察力在訓練的狀態。

關鍵是：不是「用意識叫手放鬆」，而是找到能放的那個肌肉，然後放它，讓手自己往下走。每次只動一點點，就是這樣慢慢向下。

四、拳架複習：起勢（加入新細節）

本堂再次練習起勢，強調：伸手（疏掌）出去的時候，搭配膝蓋稍微放鬆——這樣手就不需要那麼大的力量。

收肘 → 垂手 → 提腕 → 起手 → 伸直 → 屈背疏掌 → 伸手 → 成手 → 合手

一般速度和慢速各有作用：慢速是觀察工具，一般速度是熟練後的整合練習。

五、回家練習

  • 手腕拉筋：手微彎，手指左右延伸，慢慢往後翻，在緊的邊界停著呼吸。完了慢慢轉回來，放下時也要慢。
  • 放下的練習：把手舉到某個高度，停著，然後只放一點點力量，觀察它慢慢往下走的感覺。
  • 起勢複習：伸手時搭配膝蓋放鬆。

本堂提醒：拉筋是入口，不是目的。到達緊的邊界之後，重點是在那裡停著觀察，讓呼吸帶動鬆開，而不是繼續硬拉。每一次體驗到細微的變化，就是覺察力在提升的瞬間。

2026-05-14｜主題：拉筋入門——從緊中看見鬆的可能

一、這堂課真正要練的是什麼

第二堂的核心是引入「拉筋」作為另一個覺察入口。先讓身體到達「緊」的邊界，在那個邊界裡觀察呼吸與肌肉的關係，然後讓緊慢慢走向鬆。阿郎解釋：對於初學者來說，先從緊的狀態進入，比從完全放鬆的狀態開始更容易感受到肌肉的變化——因為緊的時候放掉一點點，感覺非常明顯。這就是這堂課選擇從拉筋入手的原因。

二、拉筋的邏輯：緊是入口，呼吸是工具

拉筋的目的不是把筋拉得多大，而是讓你先到達一個「緊繃的邊界」，然後在那裡停下來觀察：

  1. 這個緊和哪一塊肌肉有關？
  2. 隨著呼吸，它有沒有在微微變化？（吸氣時稍緊，吐氣時稍鬆）
  3. 停在那裡，讓吐氣的時候它再往下走一點點。

這樣一點一點地，緊繃就會慢慢打開。

緊的狀態是入口，從那裡慢慢往鬆的方向走。

阿郎特別提醒：不要再拿另外一個力量去「壓」緊的地方，不要用意識去叫它鬆，而是找到緊的感受，然後觀察呼吸的作用，讓它自己去調整。

三、手腕往後翻的拉筋動作

動作說明：

  • 手微彎放著（不打直），手指往左右延伸。
  • 手掌緩慢往後翻，一環扣一環：手掌翻帶到手肘，手肘扣到肩關節，肩關節扣到肩胛骨。
  • 翻到肩胛骨有感覺的時候停下來。
  • 加上：收下巴，後腦勺稍微往後、下巴抬一點點，讓頸椎有延展，肩胛骨那邊的感受會更明顯。

在那裡停著，自然呼吸：吸氣時緊一點點，吐氣時稍微往下鬆一點——就是在這個節律裡，讓緊的地方慢慢打開。

回來的方式：也要慢。先從手腕轉回來，然後手肘放鬆，再到肩關節，最後把手慢慢放下來。放下來的速度就是那個「幾乎感覺不到它在動」的慢。

四、放下的練習：看到肌肉的細微變化

這是本堂的核心練習之一：把手舉到某個高度，然後只放一點點支撐的力量，讓它「準備要往下掉，但又還沒有掉」。

在這個臨界狀態裡：

  • 肌肉的支撐是最少的
  • 變化是非常細微的，幾乎感覺不到手在動
  • 但你的肌肉確實知道它有一些變化在發生
你感覺不到說它真的在動，但是你知道你的肌肉已經有一些變化，這個知道就是我們要越來越熟悉的東西。

這個細微的變化，就是覺察力的訓練。越能看到這個，後面打拳的時候就越能感受到每個動作裡身體的狀態。

五、傳統站樁 vs. Feldenkrais方法

阿郎說明了傳統練法和他現在教的方法的差異：傳統站樁是站在那裡，越來越酸、越來越痛，在那個逼出來的狀態中強迫自己去感受肌肉變化——有效，但太辛苦，大家兩堂課就不來了。

他現在教的是Feldenkrais方法的精神：用慢的方式，在不痛苦的情況下，去看到肌肉的細小變化。效果一樣，但更容易持續，也更容易帶到日常生活。

慢是讓你看到肌肉在每個高度裡面所用到的狀態，不是只有看它的變化而已，看到變化之後身體就會進行調整。

六、起勢複習：增加新細節

本堂複習起勢時，阿郎加入了一個新的連結：伸手（疏掌）出去時，搭配膝蓋稍微放鬆。這樣手不需要那麼多力量推出去，動作就會更輕。

完整口訣：收肘 → 垂手 → 提腕 → 起手 → 伸直 → 屈背疏掌 → 伸手 → 成手 → 合手

速度的部分：阿郎示範了一般速度版和慢速版。一般速度是熟悉動作後的整合測試，讓學員感受自己記不記得動作；慢速才是真正的觀察工具。不要因為慢版看起來比較慢就覺得「那才是對的」——兩個都需要。

七、學員問答

Q：慢的前提是什麼？怎麼讓自己慢下來？ A：慢要有前提：先讓心靜下來。心靜下來之後能掃描身體的肌肉狀況，發現哪裡有多餘的力量，然後用意念輕輕調整，把那些「多」拿掉。做到這兩件事之後，慢才能接得起來。所以不是一開始就慢，而是先靜、先掃描、先調整，然後才慢。

Q：我感覺不到哪一塊肌肉緊？ A：可以先做拉筋，到一個緊的邊界之後，那個位置就非常有感覺。從那個感覺入手，再慢慢往鬆的方向走。如果還是完全感覺不到，就把速度再慢一倍——慢到「準備動但還沒動」的狀態。

Q：快的時候的作用是什麼？ A：快是整合性的測驗，讓身體知道它能用比較快的速度串連起來。但快的時候感覺一定比較少，所以不是用來練覺察的，是用來整合的。先由慢入手，後面再加快。

八、回家練習與下次銜接

  • 手腕往後翻拉筋：慢慢翻，在緊的邊界停著呼吸，吐氣時往下鬆一點點，最後慢慢轉回來放下。
  • 放下的練習：手停在某個高度，只放一點點力量，觀察那個幾乎看不到的細微往下走的感覺。
  • 起勢：每天練三遍，注意伸手時膝蓋微微放鬆這個連結。

下次課程繼續拉筋系列，並深化從覺察走向身體記憶的過程。

補充：關於傳統站樁與現代方法的差異

阿郎說明了他為什麼不用傳統的「苦站法」來教學。傳統站樁是站在那裡，越來越酸、越來越痛，在痛苦裡逼自己去感受肌肉——有效，但太辛苦，讓很多人沒有辦法持續。

他現在教的方式是借鑒Feldenkrais學派的精神：用慢的方式，在不痛苦的情況下，去看到肌肉的細小變化。這樣的效果和傳統練法一樣，但更容易堅持，也更容易把這個覺察帶到日常生活中去用。

最重要的是：覺察力提升之後，你對自己身體的了解就越來越深——不只是打拳，連走路、坐姿、睡姿，你都能更清楚地知道自己的狀況，也更有能力去做調整。這才是太極拳練習最大的價值所在。

拉筋練習的長期效果，是讓你的肩背的緊繃逐漸打開，讓氣血的流動更暢通。阿郎提到，打坐和站樁的過程中，他發現過敏性鼻炎的症狀在靜坐中會暫時消失，正是因為身體放鬆之後，氣血的運行改善了。拉筋的練習是在為這個打基礎——先把肩背的鎖打開，後面站樁的時候，那個放鬆才能夠真正地深入。

每一次練習，都是在加深身體的記憶——讓鬆的感覺越來越熟悉，讓緊的地方越來越容易被發現和放掉。
