一、 太極拳三大核心元素

老師將太極拳的基礎拆解為三個互相關聯的元素，並在課堂中進行整合練習：

  • 鬆緊（第一堂）：
  • 定義： 「鬆」不是絕對的狀態，而是透過比較而來的。
  • 辨識： 練習者需先感知「何謂緊」，再藉由釋放力量來體會相對的「鬆」。
  • 目標： 在身體動作中找到阻力最少、肌肉最平衡的位置。
  • 呼吸（第二堂）：
  • 氣感導引： 避免氣往頭部衝（產生壓力），應將意念留在腹部或觀想「吐氣到腳底」。
  • 壓力調節： 吸氣時身體壓力較大，吐氣時壓力變小。
  • 局部放鬆： 專注於吐氣時「壓力變小」的瞬間，可幫助特定部位（如手肘、肩膀）放鬆。
  • 動態平衡（第三堂新主題）：
  • 微小變化： 在動作過程中，保持高度覺察，讓「前一秒」與「後一秒」的身體感受差異極小化。
  • 均勻律動： 追求等速、等壓的流動感，而非斷續的發力。

二、 核心實作練習

1. 起手與落手（搭配翹翹板觀念）

  • 動作： 緩慢抬高手臂至肩高，再隨呼吸放下。
  • 觀測點： 觀察在哪個高度時，肌肉需要突然增加數倍力量才能維持？應停在力量產生劇烈變化前的高度。
  • 翹翹板原理： 抬手時，身體重心與意念要微向下沉（像要坐下的感覺），利用背部力量分擔手部的負擔。
  • 架構調整： 脊椎保持垂直、收下巴、膝蓋微彎；手不要打直前伸，應微向兩側推開，避免夾腋下導致肩部緊繃。

2. 覺知走路（貓步）

  • 腳掌接地的順序： 為了保護腳踝，建議不要用腳跟直接撞擊地面，而是先讓前腳掌或腳掌平穩著地。
  • 身體整合： 走路時保持脊椎挺直、收下巴、輕微收小腹。
  • 意念存放： 若同時留意呼吸與動作太過困難，應先將注意力單純放在「腳底與地面接觸」的感覺。

3. 肩胛骨鬆動

  • 動作： 上半身微前傾，頭部自然垂放，手部內扣轉動，使肩胛骨骨頭翻出並成為高點。
  • 目的： 透過左右輕微擺盪，讓肩胛骨縫隙的肌肉產生「鬆、緊、鬆、緊」的循環，進而達到深層放鬆。

4. 蹲姿練習（開跨）

  • 動作要點： 膝蓋必須與腳掌方向一致（呈一直線），避免膝蓋內扣造成負擔。
  • 效益： 鍛鍊大腿與小腿前側肌肉，並練習鬆開跨部。

三、 重要觀念與建議

  • 水管原理： 身體各部位（如頸椎、關節）的轉折應保持順暢，不可「折到」，否則氣與能量的流動會受阻。
  • 生活化練習： 這些元素不應只在課堂練習，而要帶入日常生活（如走路、拿手機），才能真正對身體有益。
  • 先調位、後練功： 若身體有舊傷或架構不正（如S型脊椎），應先透過特定姿勢（如嬰兒抱、蹲姿）將結構調正，否則直接練功會產生錯誤的代償。

一、課程背景與三元素整合

  • 前兩堂已建立「鬆緊」與「呼吸」的初步感知
  • 本堂導入第三元素「動態平衡」，三元素合而為一，就是太極拳的基礎
「太極拳不是武術，不是舞蹈，也不是氣功。它是一個讓你認識自己身體的方法。這三個元素，就是認識身體的三把鑰匙。」

二、動態平衡的核心定義

【定義】前一個動作與後一個動作之間的差異性要「極小」

→ 不是靜止不動的平衡，而是在流動中維持微小差異

→ 就像是水流過河道，水一直在動，但整條河的樣子幾乎不變

【為什麼要動態？】

靜止不動 → 肌肉僵化 → 緊

持續微小變化 → 肌肉有交替 → 不僵、不緊

太極拳慢動作的原理：速度放慢，才能看見每個動作之間的細節差異

「你快的時候，你看不見任何東西。你慢下來，你才看得見身體在做什麼。」

三、翹翹板原理（手起身體微蹲）

【比喻說明】身體就像一個翹翹板

手（前端）往上 → 身體（後端）往下微蹲 → 才能平衡

如果手上來了，身體不配合，重心就會往前跑 → 不穩

【實際做法】

① 手要舉起前，先讓膝蓋微微放鬆（不打直）

② 尾椎稍微往下放（像要坐下的感覺）

③ 在這個基礎上，手臂順勢往上——你會發現幾乎不需要力

④ 放下時同理，手下來，身體微微回來，重心保持穩定

「手起來不是你去抬它，是你的身體稍微蹲一點，然後手就自然往上了。這是翹翹板。」

四、重要動作要領（詳細步驟）

【抬腳練習（打破靜態僵化）】

① 站久了腳會僵，這時候用膝蓋輕輕帶起，讓關節活動

② 不是用大腿抬，是膝蓋帶動，像有一條線從膝蓋輕輕把腿拉起

③ 放下時輕輕放，不要踩踏

→ 這個練習的核心：讓原本僵化的地方重新「有變化」

→ 有變化 = 不僵 = 開始鬆的起點

【肩胛骨放鬆動作（詳細步驟）】

① 上半身稍微前傾（不要彎腰，是整個往前一點點）

② 手臂自然垂下，手掌的背面相對（不是掌心相對）

③ 讓手自然下垂，帶動肩胛骨往外翻出來

④ 左右輕輕擺盪，讓肩胛骨慢慢延展再收縮

→ 這個動作是讓肩胛骨的肌肉做「鬆緊交替」

【手臂懸掛延展練習】

① 手舉到一個高度，讓它懸掛在那裡（不主動維持，就是「掛」著）

② 配合吐氣，讓手臂慢慢沉一點點

③ 觀察：吸氣時有沒有稍微輕一點，吐氣時有沒有稍微重一點

→ 這是在練習感受細微的鬆緊差異

【蹲姿】

① 屁股慢慢往後往下，像要坐到腳跟

② 膝蓋往外擴，不要內扣

③ 起身時膝蓋也要往外不要內扣

→ 蹲的過程讓背部和腰部慢慢鬆開，不用蹲太深

五、動態平衡的生活應用

  • 走路時：每一步的落點之間，重心移動的差異是否極小？
  • 坐下站起時：是直接「跌坐」還是有感受到自己怎麼往下？
  • 拿東西時：手伸出去的那個動作，是不是有帶動到背部微微的調整？
「動態平衡不是一個練完就算的動作，它是一個你用來觀察自己身體的框架。你隨時都可以問：我現在的前後左右，是不是差不多均勻？」

六、問答重點節錄

  • Q：我試著做翹翹板，但手還是感覺很重，需要很用力。

A：代表膝蓋還沒有先鬆。你先讓膝蓋不打直，微微彎一下，再試試看舉手。通常這樣做，手就輕了。

  • Q：肩胛骨放鬆那個動作，我感覺在動的是腰，不是肩胛骨。

A：很好的觀察。這代表你肩胛骨的部分還比較難獨立感受。先讓腰動也沒關係，慢慢地肩胛骨會開始有感覺。

七、課後練習建議

  • 「放慢一倍」：今天選一件日常事情（走路、洗碗、開門），速度放慢一倍來做
  • 抬腳練習：站久後做 5-10 次膝蓋輕輕帶起再放下，感受僵化在哪裡
  • 肩胛骨鬆動：早晨起床後做 1-2 分鐘，讓肩頸不緊繃一整天
  • 蹲姿：每天 1-2 次，不必蹲很深，感受背腰的放鬆就夠了
